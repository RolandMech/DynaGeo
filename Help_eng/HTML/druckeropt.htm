Printer options...

This dialog allows you to configure the print output of DynaGeo:


In the field Output format you can select the output orientation and range of the printer. There are 3 different items:

  • Portrait, upper half of page
  • Portrait, full page
  • Landscape, full page


In the field Output window you can set some details about the output window:

  • The scaling factor contains the current factor by which one cm on the logical screen is multiplied before printout. E.g. if the scaling factor is 0.666, a line segment of 10 cm on the logical screen is printed out as a line segment of 6.66 cm.
  • The relative border width contains the relative width of the margin in percent of the current print area. In landscape format the print area is the entire sheet; in portrait format only the upper half of the sheet. E.g. if the relative margin is 5, then the printout width is (100 - 2x5)% = 90% of the total width or height of the printable area.


In the field Active printer the actually selected printer of your Windows system is shown.

  • If you click the button "Configure/change printer", the normal Windows printer management dialog appears. There you can change the printer if there is more than one printer installed in your Windows. And you can configure the active printer.


On the right side of the dialog you find 4 buttons:

  • "Preview" calls a window showing a print preview.
  • "Print" closes this dialog and starts the printing.
  • "Close" closes this dialog and saves your changes without printing
  • "Abort" closes this dialog and resets your settings to the state before the dialog was called.
