List of the ALT commands


Some of the DynaGeo commands don't have an own button in the toolbars. Nevertheless, they can be reached via the toolbars: just click an appropriate toolbar button while holding down the ALT-key. Please note that these "ALT commands" cannot be repeated automatically, even if the main command of this tollbar button is. There is no use in pressing the "ALT" key and the "Shift" key together while clicking on the button!

Here is a list of the available "alternative" commands:


on the "Main bar":

  • ALT + :     Show all hidden objects



on the "Construct bar":

  • ALT + :     Lattice point


  • ALT + :     Perpendicular segment from a point onto a line


  • ALT + :     Circle through 3 points


  • ALT + :     Circle with radius from a segment


  • ALT + :     Radical axis of two circles


  • ALT + :     Bisector of the exterior angle


  • ALT + :     Ray of determined angle


  • ALT + :     Regular polygon



on the "Curves bar":

  • ALT + :     Riemann sum of the area below a function graph


  • ALT + :     Normal line to a curve



on the "Measure & Calculate bar":

  • ALT + :     Measure the length of a line segment
