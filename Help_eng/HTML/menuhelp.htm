Help Menu

Help Menu

Here you can access the online help of DynaGeo. This menu also offers additional information concerning the license conditions as well as program registration possibilities.


Contents:

    Displays the list of contents for the EUKLID DynaGeo help

Help about the actual command:

    When this command is chosen, a help page for the actual command is displayed. If there is no command active at the time (i.e. if you are in drag mode), then the help contents is displayed. The usual short cut for this command is implemented too: just press the F1 key!

Basics:

    Leads to a text which explains the basic ideas of the EUKLID DynaGeo program.

The EUKLID DynaGeo window:

    Explains the external appearance of the EUKLID DynaGeo program.

Keyboard and mouse:

    Gives an overview of the possibilities to control the EUKLID DynaGeo program.

License conditions:

    Displays the license conditions.

Registration:

    This command only appears in the Shareware-version. It opens a window in which a licensed full version can be made from a shareware version by entering the path to a valid EUKLID DynaGeo license file.

About EUKLID DynaGeo:

    Calls the title window in which the version number, program license data etc. are displayed.
