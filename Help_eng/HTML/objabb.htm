Map object by...     

(automatisch wiederholbar)

This command maps an object by an already existing mapping.

When you call this command, a list will appear containing all the mappings that have been previously defined in the actual drawing. Choose the desired mapping, then click on the object you want to map. The mapped object is created and displayed.



Please note:

  • If the drawing does not yet contain any mapping (e.g. in an empty drawing) then this command is not available: both the menu item "Map object by..." and the corresponding tool button on the "Mappings" bar are deactivated und appear grayed.

  • The list of the available mappings does not only contain those mappings created via the command "Define an affine transformation", but all mappings that have been previously executed in this drawing.
