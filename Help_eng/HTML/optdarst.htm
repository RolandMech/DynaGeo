Presentation options

Options | Presentation


Standard point shape:
    Here you decide about the shape of the different point types (basis points, fixed points, constructed points). When you click on the small label showing the actual point shape, a list of possible point shapes appears where you can choose from.

Standard line styles:
    Here you decide about the style of the different line types (simple lines, locus lines). When you click on the small label showing the actual line style, a list of possible line styles appears where you can choose from.

Point size:
    This parameter controls the size of a point on the screen. The value is the radius of a circular displayed point in pixels.
    Range: [ 2 ........ (5) ........ 10 ]

Standard colour of the co-ordinate system:
    This parameter controls the colour of the co-ordinate system in a new drawing. Normally, it is displayed in a light gray.

Name colour follows object colour:
    If this checkbox is checked, the colour of any object’s name will always be the same as the colour of the object itself. If not, the two colours are independent of one another.

Mark right angles with a point:
    If this checkbox is checked, then a right angle is marked with a point in the angle arc. Normally, a right angle is marked with a small square instead of the normal angle arc.

Show angle marks filled:
    If this checkbox is checked, the angle mark is filled with a colour. Normally, a new angle appears yellow, but if you change the colour of the angle arc, the colour of the filling is adjusted.

Enclose measures with a frame:
    If this checkbox is checked, any measure label is enclosed by a dotted frame, if not then the frame is ommitted.

Fill new polygons on creation:
    When you choose this option, every new created polygon will be filled. The colour and the pattern of the filling will be chosen automatically, and DynaGeo will try to use a combination not used before for a filling in this drawing.
    If this option is unchecked polygons are created without a filling. Then they can be filled later.
