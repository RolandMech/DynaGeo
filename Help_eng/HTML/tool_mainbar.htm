Main Bar

The Main bar contains the following icons:

    New drawing
    Load drawing
    Save drawing
    Print drawing

    Undo last construction step
    Redo last undo
    Name object
    Hide / show object
    Delete object

    Snap point to a line
    Unsnap point from line

    Load a picture
    Create a new textbox

    Repeat last macro

    Check correctness of construction

    Show / Hide setsquare
