Parameters of animation    

This dialogue allows to edit the parameters of an animation. The following items are available:


Steering object:
    Here you find a list of all possible steering objects of your drawing. Just select the desired one. Only then, the other parameters of the animation are available.

Lower bound and Upper bound:
    The animation is controlled by a position parameter. These two fields show the bounds of the interval that holds all the possible values of the position parameter. Normally, these bounds are set to standard values. Only if you use a number object to control the animation, you can edit these values.

Param step:
    Here you choose the increase of the position parameter with every animation step. This controlls the speed of the animation. For further details see The basics of aminations.
