Name object    

(automatically repeatable)

Click the object you wish to name, and rename it in the opened dialogue box. Every object is automatically given an original name comprised by a letter for the type of object (e.g. "P" for point) and a serial number. You are however allowed to type any name, but EUKLID DynaGeo checks if the name contains invalid characters or if it is perhaps already in use. In any of these cases you get an appropriate message.


Object names must obey the following rules :

  • Allowed characters are lower and upper case letters and the digits.

  • A name must not contain a blank. If you want to seperate names into pieces, please use the underscore ("_") or the rhomb ("#"). However, a name must not start with any of these characters.

  • Forbidden characters are: all punctuation marks, mathematical operators (like "+", "-", "*", "/", "^", "&", "\") and all brackets.


The standard font is preset to "Arial 12pt", but you can change this in the "Options" on the register page "Texts".

When naming an object, you can choose from all of the installed fonts. The font size is freely adjustable too. Moreover, you can use all the usual text attributes like bold, italic, underlined, subscript and superscript. Note that a click on an attribute button affects only the actually selected text!

If you leave the dialogue box by clicking OK, the new name is given to the object. If you leave the window by clicking Cancel, the object will keep its old name.
