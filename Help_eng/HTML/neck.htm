Polygon    

(automatically repeatable)

Click with the mouse to mark the vertices of the polygon you wish to draw. You may either use already existing points or draw new ones with the mouse.

You close the polygon by either clicking once again on the first defined vertex of the polygon or by double clicking when you create the last defining vertex.



ALT variant: Regular polygon

    If you start this command by clicking the tool button on the Construct bar, and you hold down the ALT key while you click, a regular polygon is created instead of a general one. For the details about the parameters see the description of the command Regular polygon.
