Hot keys:

F1 key :

    Calls the help text for the current command. If the key is pressed while processing a menu command, then the help text for that menu command will be shown. If the key is pressed while no command is being carried out, then the help contents page is called.


Ctrl O :

    Opens a drawing
    (short-cut for the menu command File | Open)


Ctrl S:

    Saves the present drawing
    (short-cut for the menu command File | Save)


Ctrl P :

    Prints the actual drawing
    (short-cut for the menu command File | Print)


SHIFT key :

    If you hold down the SHIFT key while clicking on a toolbar button then the corresponding command will be repeated automatically until you stop this mode by pressing the ESC key. The automatic repetition will only work with suitable commands. For details see the description of the individual commands.


ALT key :

    For some commands, there is an alternative variant: if you hold down the ALT key while clicking on a toolbar button then the corresponding alternative command will be executed. For details see the list of ALT commands.


Ctrl R :

    Repeats the last command (if a suitable command is stored)


Ctrl U or F5 key :

    Updates the screen. Re-draws the current drawing.


Ctrl D :

    Diminishes the drawing
    (short-cut for the menu command Reduce drawing)


E key :

    Enlarges the drawing
    (short-cut for the menu command Edit | Drawing... | ...enlarge).


B key :

    Makes all basic objects blink.


ESC key :

    Breaks off any action and returns to the drag mode.


ALT F4:

    Closes the EUKLID DynaGeo program.
