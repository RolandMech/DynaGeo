Circle from centre and peripheral point    

(automatically repeatable)

With the left mouse button, first click to mark the centre of the circle and then on a point to define its periphery. The points do not have to exist already. If you click at an empty spot of your drawing a new point will be created there. So the circles produced by this command will always have a visible centre point and a visible point on the periphery.



ALT variant: Circle through 3 points

    If you start this command by clicking the tool button on the Construct bar, and you hold down the ALT key while you click, then 3 parameter points are expected. Finally, the circle running through these 3 points is created and displayed. In case of 3 collinear points, this circle has an "infinite" radius and degenerates to a straight line.
