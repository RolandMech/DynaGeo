Point

Point    

(automatically repeatable)

Clicking the left mouse button at any position in the working area produces a point. The initial shape of such a basis pointis a filled black square.

In addition to the standard items the context menu of such a basis point holds the following commands:

  • Snap point to line... respectively Unsnap point from line :
    When the point is "free", the command Snap point to line can bind the point to a straight line, a circle, a polygon or a locus line. When it is already bound, the command Unsnap point from line will release the binding and change the point back to a free point again.

  • Steps... :
    This command will bring up a window where you can specify how the point will move when dragged by the mouse:

    • most smoothly, i.e.:
      the point follows the mouse pixel by pixel;

    • in jumps of adjustable width, i.e.:
      while following the mouse the point will jump to the next point of a (virtual) grid. In this case you can specify the step width (i.e.: the distance of the grid lines). Setting the width to 1 will force the point to always have integer co-ordinates.

    This menu item is only available for unsnapped basis points.

  • Fix :
    This commands converts a basis point to a point with fixed co-ordinates.



ALT variant: Lattice point

    If you start this command by clicking the tool button on the Construct bar, and you hold down the ALT key while you click, then a lattice point is created instead of a basis point. A lattice point is a special basis point that has always integer co-ordinate values. This is achieved by setting the "steps" parameter of this point to the value "1.0" when creating such a point. It can be dragged by the mouse, but instead of following smoothly, it just jumps from one lattice spot to the next.
