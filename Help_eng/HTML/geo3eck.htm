Show / hide set square    

This command shows a transparent set square that lies on your drawing. The next call to this command will hide the set square again.

You can drag the set square with the mouse like a normal base object. When you click inside the circle line that shows the angle measures and drag the mouse, you can move the set square. When you take it outside of this circle line (in the corners) you can rotate it around its centre. If your computer is a bit slow, it may be necessary to perform this manipulation carefully and not too fast.



Please note:

  • The set square cannot interact with any of the other objects of the drawing. It is purely a decoration of the drawing, serving only for demonstrations.

  • The set square is only available if your drawing uses an isotropic co-ordinate system. If you use different unit lengths on the two axes, it is hidden automatically.

  • To show the set square is a rather expensive task for Windows and needs a lot of system resources. So you should only show the set square as long as you really need it, and hide it immediately afterwards.

  • To achieve a smooth dragging of the set square, it may be necessary to activate the internal "double buffering" of DynaGeo (see "Options | Internals").
