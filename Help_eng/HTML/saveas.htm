Save as...

You may type in the new file name in a dialogue window. In order to inform you, a list displays the already existing names of saved .GEO-files. You may choose one of these names or enter a new name into the edit field File name. When you confirm (with Ok) the current drawing is saved to a file with this name. If such a file already exists, you are asked to confirm if you really want to overwrite it.

The title line now displays the new file name as the name of the current drawing.

From this dialogue window you may also save an EUKLID DynaGeo drawing as a BMP, WMF or EMF file. Choose the option BMP, WMF or EMF in the file type selection box at the window's bottom. A list of the previously saved BMP (or WMF or EMF) files will be shown. If you have selected part of the drawing in a selection frame before opening the dialogue window, only this part is saved to the graphic file.
