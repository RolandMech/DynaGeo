Options | Export

This page contains 3 sections:


1) Options for print output:
    Generate line style internally
    Generate circle style internally
    Allow dashed fillings
    Allow rasterized fillings
      These options are for older printer models with poor printer drivers that cannot print dashed or dotted lines or dashed or rasterized fillings. It may take some experiments before you find out what's the best configuration for your printer.

2) Options for clipboard export:
    Generate line style internally
    Generate circle style internally
      The first two options control where line styles are created.

    Dilation factor in x-direction
    Y/X aspect ratio
      The next two parameters are for specialists who want to control the size and aspect of the clipboard export very precisely.

3) Options for bitmap export:
    Resolution of the image file
      This parameter sets the dpi value for the export of images out of DynaGeo. Normal copies of the screen content show only the poor screen resolution and are of low quality especially when printed out later. To avoid this, you can choose a higher resolution for the export into pixel-oriented graphic files. This supports the formats BMP, PNG and JPG. The highest value is 1200 dpi, but be careful: this will result in rather big graphic files!
