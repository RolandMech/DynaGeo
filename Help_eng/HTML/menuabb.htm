Mappings Menu

This menu contains commands to map objects. The mappings provided are the congruences (reflection in a line resp. in a point, translation, rotation) and the dilation (scale change). Moreover, there is the reflection in a circle as an example for a mapping that does not preserve straight lines. Finally, the full range of the affine transformations is provided.
