Edit transformation matrix...

View/edit transformation matrix...

Calling this commandshows you a list of the already defined affine mappings where you can choose the transformation wanted. Then a window appears where this transformation is shown in matrix form.

If the transformation is an affine transformation with coefficients previously entered by the user, then you can edit the values of the coefficients. In all the other cases, the coefficients can only be displayed but their values cannot be changed. If e.g. you defined a reflection in a line g, the coefficients are determined by the position of this line and there is no use to change their values manually.


Please note:

  • If your drawing does not contain any mapping (e.g. in an empty drawing), then this command is not available: the correponding menu item is deactivated and appears grayed

  • The list of the available transformations does not only contain the mappings defined by the command "Define an affine transformation..." but also all other transformations used previously in this drawing. Only reflections in a circle are left out for these mappings cannot be represented in matrix form.
