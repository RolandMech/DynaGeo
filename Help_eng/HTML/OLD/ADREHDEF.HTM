Mappings | Define a rotation    

This command serves to set the angle of rotation:

Indicate

  1. an original point,
  2. the centre of the rotation and
  3. the corresponding rotated point resp. a direction point that defines the ray from the centre through the rotated point.
The point indicated last will be dragged to the correct distance from the centre, if necessary. Then the arc around the centre is drawn that runs from the original point to the rotated point.

This commands is equivalent to the command "Arc". It is added here to avoid repeated changes between the variuos toolbars.
