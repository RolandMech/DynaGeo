File | Properties...

This menu item leads to a window that shows the properties of the currently loaded GEO file, e.g. the date of creation of the file, the date of the last changes and the EUKLID DynaGeo version that was last used to edit this file.

Moreover, if you hold an EUKLID DynaGeo licence, you can add a copyright note to your EUKLID DynaGeo GEO files. Nobody but you can edit and change this copyright note. When you load a GEO file copyrighted by someone else, the copyright edit window is read-only.


Attention !

    Your copyright note does not write protect your GEO file! Every user can edit his copy of your drawing, change your constructions, delete some of your objects and add some other... Only the copyright note itself is protected against alteration!
