Problems with the HTML help

The new Microsoft HTML help system depends on an Internet Explorer 4 (or newer). From WIN98SE on, there should not be any problem.

With earlier WINDOWS versions, there may be problems with the HTML help. In most cases it is sufficient to install an update of the HTML help functions named "hhupd.exe". You can get this file from Microsoft's Internet pages (www.microsoft.de resp. www.microsoft.com). Alternatively, you can update your Internet Explorer to an actual version.
