Automatische Auswahl

Wenn Sie im Editier-Modus an eine Stelle des Bildschirms klicken, an der sich mehrere Objekte befinden, bemüht sich EUKLID DynaGeo herauszufinden, welches der Objekte Sie nun wohl meinen. Dazu ist jeder der verschiedenen Objektarten eine charakteristische Priorität zugeordnet, und zwar gemäß der folgenden (nach abnehmender Priorität geordneten) Reihe:

Punkte / Strecken / Geraden / Kreise / Namen / Maßobjekte / Polygone.

Kann EUKLID DynaGeo nach dieser Reihe zweifelsfrei entscheiden, welches Objekt gemeint ist, dann wird ohne weitere Nachfrage dieses Objekt bearbeitet. Steht der Auswahlcursor jedoch z.B. so, daß 2 verschiedene Strecken gemeint sein könnten, aber kein Punkt, dann fragt EUKLID DynaGeo mit einer Auswahlbox nach, welches Objekt Sie nun tatsächlich bearbeiten wollen.

Beim Erzeugen und Löschen von Objekten ist die automatische Auswahl abgeschaltet, damit EUKLID DynaGeo nicht aufgrund eines Mißverständnisses ein verkehrtes Objekt erzeugt bzw. das verkehrte Objekt löscht.
