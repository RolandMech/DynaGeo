Remove a macro

If more than one macro is loaded a list is displayed with the names of the macros. Click on the macro you wish to remove, and the macro is deleted from the macro menu and from RAM.

If there is only one macro active when you apply this command, this one is deleted at once.


Please note:

    If a corresponding macro file exists, this file is not deleted by this command. If you wish to delete a macro file you can use the WINDOWS' file explorer. Macro files carry the extension *.MAK.
