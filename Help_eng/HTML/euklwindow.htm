The EUKLID DynaGeo window

EUKLID DynaGeo is a simple WINDOWS program than can be run either in a window or as a full screen application. The window's Title bar shows the program title followed by the name of the EUKLID DynaGeo file (*.GEO) currently displayed. If the drawing board is still empty or the current drawing has never been saved before, then the filename NONAME.GEO is displayed.

Below the title bar there is the Main menu. Every one of its entries leads to a pull-down menu, which unrolls when the appropriate Main menu entry is clicked. Under the Main menu the Tool bar is displayed that allows a comfortable access to the commands most often used. The command icons are distributed among several tool bar pages that cover different tasks or working situations.

At the bottom of the window appears the Status bar where the current status of the program is shown. Additionally the current cursor position is shown to the left of the status bar. When you apply a command, the Status bar always shows hints about the actually expected input data for this command.

The area between the toolbar and the status bar contains the actual drawing board, where you can construct your drawing.
