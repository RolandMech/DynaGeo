Reflect an object in a point    

(automatically repeatable)

Indicate the object and the centre of the reflection.

The corresponding reflected object is created and displayed. Each defining point of the original object is connected with its image point by a dotted trace line, as long as you did not deactivate the "automatic trace generation" in Options | Mappings.
