Copy

This command copies the current contents of the EUKLID DynaGeo window to the clipboard. So you can import EUKLID DynaGeo drawings into other WINDOWS programs.

If before copying you have made a selection frame on the drawing, then only the framed part of the drawing will be copied.

The drawing is copied to the clipboard in two formats:

  1. in BMP format (i.e. as pixel graphic)
  2. in EMF format (i.e. as vector graphic)

When you import an EUKLID DynaGeo drawing from the clipboard into e.g. WORD, then WORD will insert it as a EMF image in vector format. This means that after inserting the image with a positioning frame at the right position in the text, you may change the size of the image without reducing its quality. You should not however try to re-work the drawing with MSDraw. This program has only limited drawing capabilities, and does not understand (for example) the arcs used in EUKLID DynaGeo.

When you export an EUKLID DynaGeo drawing, a BMP version is also written to the clipboard. Usually, the resolution of this copy is equal to the normal screen resolution, and so later printouts may look poorly. To avoid this, you can increase the resolution of DynaGeo's pixel graphic export: just set the "resolution of pixel graphic exports" to a higher value. You find this parameter via the menu item "Miscellaneous | Options | Export".
