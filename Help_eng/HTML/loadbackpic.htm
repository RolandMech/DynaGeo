Load a picture    

A standard file dialogue window is opened and you can select a graphic file. The supported file formats are:

    bmp, jpg, jpeg, wmf, emf and png.
The window shows the graphic files that have been found in the current folder. You can also navigate to another folder or change the drive to get access to graphic files stored in other places. When you click the Okay button, the selected graphic file is loaded and will be used as a background picture for your drawing.


The context menu of a such a picture holds the following entries:

  • Hide picture :
      This commmand hides the picture. It works just like the "Hide" command of any other object.

  • Show frame / Fix picture :
      Normally, a background picture has a fixed extension at a fixed place. After loading, the picture is located in the middle of the actual window and displayed without a frame. In this state, you cannot change neither the size nor the position of the picture.
      When you select the command "Show frame", a gray frame is drawn around the picture. In this state, you can drag the picture as a whole, and you can change its height and width by dragging the frame lines. If you want to change the size without changing the aspect, you can drag the corners of the frame.
      When the picture has the right position and the desired extensions, you should carry out the "Fix picture" command. This will hide the frame and so protect the picture against accidental manipulations.


Please note:

    The graphic files are not just linked into your drawing but embedded. So subsequent changes to the original files have no effect on the pictures imported in your drawing.
