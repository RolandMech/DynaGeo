Show all hidden objects

With this command all hidden objects of your drawing will be made visible. This step cannot simply be revoked! So EUKLID DynaGeo askes you, if you really want to do this. The command is executed only after your additional confirmation.

If your drawing has a hidden co-ordinate system, it will stay hidden. The command has only an effect on the user defined geometrical objects.

If at a later time you want to hide again all the objects that became visible by this command before, you have to hide them one by one! Okay, that's a bit laborious, but the automatic repetition of the Hide object command may help you with the work!
