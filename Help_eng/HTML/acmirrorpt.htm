Reflect an object in a circle    

(automatically repeatable)

Indicate the original object and the circle in which the object is to be reflected. Then the reflected object is created and displayed.


Please note:

  • This command is restricted to map only points, lines and circles.

  • If you let repeat this command automatically, all subsequent reflection will use the circle of the first one. So all original objects are reflected in the same circle!

What's that, a reflection in a circle?

    The reflection in a circle is not yet equally known as the congruent transformations and the similarity transformations. So here is a definition: if P is to be reflected in a cirle c(M,r) with centre M and radius r, then the image point of P is that unique point P' on the ray [MP), that satisfies the equation: |MP| * |MP'| = r².

    The reflection in a circle is a rather interesting transformation, because it does not preserve the straight line property as we are used when looking at congruent or affine transformations. Instead it maps circles to circles! (In this context, straight lines can be seen as circles with infinite radius.) Only the lines running through the centre of the reflection circle are mapped to straight lines - they are even fix elements of this reflection. All other straight lines are mapped to circles. Just play some time with this command to get used to the secrets of this somehow strange mapping!
