Animation Bar

The Animation Bar

This toolbar is only visible if your drawing contains a valid steering object for animations. Otherwise, the toolbar is hidden. If it is visible, it contains at least the following icon:


    Edit parameters of animation

    A click on this button opens a window where you can select the steering object for your animation and set the speed control parameters.


    When you have selected a steering object, the navigation command buttons are shown too:



    These commands work just like the corresponding buttons of a cassette recorder: the two green arrows start the animation (backward and forward), a click on the red button in the middle stops it, and the outer buttons let you jump to the start or the end of the animation.
