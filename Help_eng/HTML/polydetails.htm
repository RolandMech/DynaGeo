Details about polygons

A polygon is defined to be the series of segments surrounding this area: the polygon actually is the border of its area!

When you create a polygon (applying the command Triangle or Polygon), its side segments are created too, if necessary. But even if you delete all of the line segments that make up the sides of the polygon, the polygon itself will remain a valid object. Still you can snap a point to it, and you can measure its enclosed area.

If you want to display a polygon filled, you can apply the command Fill area. If you want all polygons to be filled when created, activate the option "Create polygons filled" on the tab sheet Presentation of the Options.
