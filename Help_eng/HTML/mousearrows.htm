Mouse cursor shapes

Mouse cursor shapes:

A particular shape of mouse cursor goes with every program state:


The arrow is the normal mouse cursor form. With this cursor you can choose the menu commands and make choices within the dialogue boxes.

The cross hairs is displayed as an input mouse cursor. It is always displayed when the program expects an input (e.g. choice) of a geometrical object.

The cross hairs with circle is displayed when the input mouse pointer is on an object that fits the expected criteria.

The pincers  are displayed in the dragging mode, as soon as the mouse pointer is placed on a basis object or while a basis object is being dragged.

The hand is displayed to change the position of object names and measurements, as well as to move the entire drawing.

In addition, the usual WINDOWS egg-timer is displayed while a time consuming operation is being carried out, and you must wait a while. With EUKLID DynaGeo, this is (hopefully) only necessary during printer output.
