Options | Option data file menu

The local menu of the Options window contains the following six items:

  • Load default settings
    This commmand resets all options to their standard values.

  • Load user settings
    This command allows you to reload your personal settings. Normally, this is done at every program start automatically. But if you change your actual configuration (without saving it!) this command allows to return to the settings that were saved last.

  • Save actual user settings
    This command allows to save your actual settings. They will be reloaded automatically at the next start of the program in order to restore the program state of your last DynaGeo session.


The last three menu items are only for DynaGeo administrators. A DynaGeo administrator is a user that has write access to the central configuration file DYNAGEO.INI in the DynaGeo directory. "Normal users" can only read this file. So, the following menu items appear disabled.

  • Save actual user settings as standard
    This command lets the administrator save his personal settings as standard settings for all users. The administrator can force the users to work with these standard settings by blocking the use of personal configuration data in the Network options.

  • Actualize the license data
    This command allows you import new license data e.g. after an upgrade of your license. Actually, the license data file is copied to the DynaGeo directory, and so you must have write access there! Normally, there is no need to use this command.

  • Network options...
    This command opens a dialogue where the administrator can prescribe if the other users can work with their personal configuration or if they are forced to take the standard settings. For further details see the page about Network options.


Normally, your personal settings are saved automatically whenever you close the program. So you may hardly feel the need to use the first three menu items. The last three menu items should also be used only if you know exactly what you do.
