Circle with determined radius    

(automatically repeatable)

Upon selecting this menu item, a small window is opened. There you enter the radius of the circle. Its value may be a constant or an expression. Then you click on the screen to mark the centre of the circle.

The circle is created and displayed.

Beside the standard items the context menu of a circle with determined radius shows an entry named

  • Radius...,
    that allows you to edit the radius' value (or expression) afterwards.



ALT variant: Circle with radius from a segment

    If you start this command by clicking the tool button on the Construct bar, and you hold down the ALT key while you click, then other parameter objects are expected: first click on an already existing segment and then on a point. Finally, the circle around this point is created and displayed that has a radius equal to the length of the segment.
