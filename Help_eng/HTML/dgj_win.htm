Does DynaGeoJ change my Windows?

    DynaGeoJ is a normal JAVA-Applet that is managed automatically by WINDOWS itself. Whenever the browser is to show a HTML page that needs DynaGeoJ, the applet is downloaded according to the data of the <APPLET>-Tag in the HTML text of the page, either from the web if there is an URL provided, or from the local file system if there is only a relative path. The applet is started and loads the drawing from the GEO file given in the "param" tag inside the <APPLET>-Tag. Finally it displays the drawing in a window with the desired extensions "width" and "height".

    Though during the installation of DynaGeo on your computer one version of the DynaGeoJ applet is put into the "DynaGeo\Viewer" folder, in most cases the browser will perform a new download of the JAR archive from the web. How the browser gets a runnable instance of the DynaGeoJ applet is decided only by the data in the <APPLET>-Tag.
