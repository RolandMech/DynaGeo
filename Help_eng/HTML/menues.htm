The Main Menu


The menu system of EUKLID DynaGeo gives access to all implemented commands except those on the "Shape&colour" bar. As long as you did not choose any special menu configuration all of the implemented commands will be active and available.

The Main Menu contains the following items:

    File Menu

    Edit Menu

    View Menu

    Construct Menu

    Mappings Menu

    Measure Menu

    Macro Menu

    Miscellaneous Menu

    Help Menu
