Build a number object

Create a number object    

A number object is a slide control that implements a number. The number's value can be changed in a predefined interval, by sliding the control knob of the number object. With the help of number objects you can realize an adjustable dilation factor or a parameter to make a point move on a certain path.

When you choose this command, a number object is created, filled with standard data and displayed. It is automatically positioned at a free spot of your drawing. In the upper half of the number frame, the upper and lower bound of the interval and the actual value are shown. Below appears the slider.

The interval is initially set to [-3; 5]. You can change these settings with help of the small "up" and "down" triangles next to the bounds' display. The actual value is initially set to 1. This value can be changed with the slider: just drag its knob.

You can use the actual value of a number object Z1 in the expression of a term object by inserting the name "Z1" of the number object in the expression of the term object. For example, if you enter the expression "3*Z1" in the term object, then this will always show 3 times the value of the number object Z1. For further details see the page about The syntax of terms.


Beside the usual items, the context menu of a number object contains another command:

  • Edit range:

    If you choose this entry, a window opens where you can edit the interval bounds and the actual value directly.

    In addition, you can enter a "Step width": this is used to force the number object to change its value only in steps of this given width. Thus, the value of the number object is "quantized". If for example you set the step width to "1", the number object's value will always be integer. If you set the width to "0.2" its value will always be a multiple of "0.2".
