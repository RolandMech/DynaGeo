Dynamic drawings in HTML documents

EUKLID DynaGeo allows to display dynamic drawings in HTML documents. So the reader of your document can drag the objects of the drawing, even if EUKLID DynaGeo is not installed on his computer at all. This can be achieved in two different ways:

  • You use the DynaGeoX viewer supplied with DynaGeo and based on the ActiveX technology of Microsoft.

  • You use the DynaGeoJ viewer supplied with DynaGeo and based on the JAVA technology.

Before studying the further comments, please note that DynaGeoX urgently needs the Microsoft Internet Explorer (version 4 or newer). If you want to develop for other browsers too, you'd better use DynaGeoJ.
