Reset drawing to original size

This command resets the drawing to the state in which the co-ordinate system has the same scaling and position as in an empty new drawing: the unity length on each axis is 1,00 cm (logical WINDOWS cm!), and the origin is in the middle of the drawing window. Then the whole drawing is adjusted to this new co-ordinate system. Only number objects, term objects and (unbound) text objects remain at their old positions: these objects are not tied to the co-ordinate system but to the upper left corner of the window.
