Line segment

Line segment between two points    

(automatically repeatable)

Click with the left mouse button on the two end points of the line segment required. The segment will be created and displayed.

You may choose already existing points as end points. If you click on an empty area, a new point will be constructed. So a line segment will always have a start and an end point.
