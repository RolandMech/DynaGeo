Line segment of fixed length    

(automatically repeatable)

Upon choosing this menu item a small window opens in which the length of the line segment (in cm) may be typed.

Then, using the mouse, click first on the start point of the line segment and next on a second point to define its direction. If this "direction point" is moveable, it is automatically dragged to the correct distance from the start point. If it is not moveable however, a new end point is created at the correct distance.
