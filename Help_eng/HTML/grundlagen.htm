Basics

DynaGeo is a geometry drawing program that produces dynamic drawings. You can choose which geometrical object you wish to produce, using the menu commands or clicking the appropriate symbol on the toolbar, and enter the data for that object. DynaGeo then draws that object on the screen.

If you have produced multiple objects you can change the drawing by dragging a basis object with the mouse. All objects the position of which depend on that basis object’s position are then automatically updated so that the drawing looks as if it had been originally drawn in the new position.

DynaGeo contains a powerful context sensitive help function: if you don't know how to go further, press the F1 key from anywhere within the program, and DynaGeo will show you a help for the actual situation.

DynaGeo drawings may be saved and loaded as well as exported via the clipboard and of course printed. The data file format used is a special one, and data files carry the standard extension GEO.

You may measure the present distance between any objects as well as the angles. Additionally, you may enter formulas or expressions in which these measurements are used as variables. The value of these variables are also updated when you drag an object and the new values displayed.

If a number of construction steps should be carried out several times in the same way, you can create a macro, and thus expand the abilities of EUKLID DynaGeo. Macros can be saved and loaded. The presently active Macro commands are automatically displayed as additional menu items in the Macro menu of the DynaGeo window.

After a construction has been made, it is possible to replay and see the sequence of the construction steps again on the screen, just like a film. You can add comments to your drawing by using a textbox object. So you can give some hints to your user or present a problem to be solved. If required, you can display a construction text in an extra window.

Every DynaGeo drawing contains a Cartesian co-ordinate system with a unit length of 1 cm. This allows to produce points with given co-ordinates. When a drawing without co-ordinates is preferred the co-ordinate system may be hidden.

DynaGeo offers many construction commands. But you can also limit the commands available by choosing different command sets from a configuration menu. So a teacher can make the students construct a perpendicular bisector using elementary basic tools (i.e. compass and ruler) instead of the perpendicular bisector command. You can even create your personal command sets.

A DynaGeo drawing can theoretically consist of as many objects as you can fit into your computer’s RAM. With drawings of this complexity, however, the dragging process can be somewhat laborious. The required calculations and (especially) graphic outputs need a fast computer fitted with a speedy graphic card.
