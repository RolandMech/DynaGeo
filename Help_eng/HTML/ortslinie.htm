Record a point's locus line    

Click on the point the locus line of which you want to record. This point is hereafter called the "producing point". Then drag any of the basis objects the producing point is depending on. This point is called the "dragging point" of the locus line. When now the producing point moves due to the moving of the dragging point, a series of marks is set indicating the producing point's former positions. When you release the mouse button to end dragging, the locus line is drawn as a smooth line through all of these recorded positions.

If you want to record several locus lines simultaneously, hold down the Shift key while clicking the first producing point. This indicates to the program that several locus lines are to be drawn. A window is opened with only an Ok, done! button. Now click (without Shift!) on all the points which should leave a trace. Finish by clicking the Ok, done! button. Finally, by dragging the common dragging point the selected producing points all leave a trace (if moving the dragging point also causes the producing points to move).

The dragging point can be substituted by a number object: instead of dragging a basis point you can drag the control knob of a number object, if only the co-ordinates of the producing point depend on that number object's value.


Please note:

  1. New locus lines normally are created as dynamic locus lines if possible.

  2. If you want to create only a static trace instead of a dynamic one, please press the ALT key while clicking the producing point.

  3. Beside the standard items the context menu of a locus line may show the following commands:
    • Thick line or Thin line:
        to switch the line's width;
    • Show as series of points or Show as line:
        to switch the presentation style;
    • Dynamic:
        to toggle the dynamic mode;
