Mouse capture area:

    This parameter controls how close the mouse pointer must be to a geometrical point before it is captured. The capture distance is measured in pixels.
    Range:&xnbsp;&xnbsp; [ 1........ (5) ......... 10 ]

Zoom factor:

    This is the "stretching" factor by which the picture is expanded or reduced when a zoom command is applied.
    Range:&xnbsp; [ 1 ...... (1.5) ....... 10 ]

Tolerance for angle comparison:

    An angle w1&xnbsp;is taken as being the same as another angle w2 &xnbsp;when w1 &xnbsp;lies within the interval [w2-dw, w2+dw]. Here, the dw value in degrees can be set.
    Range:&xnbsp;&xnbsp; [ 10-5........(0.5)........10 ]

Tolerance for distance comparison:

    A distance d1&xnbsp;is taken as being the same as another distance d2 &xnbsp;when d1 &xnbsp;lies within the interval [d2-dd, d2+dd]. Here, the dd value in pixels can be set.
    Range: [ 1*10-20.......(1*10-8).......10 ]

Point command with extended capabilities:

    When you check this checkbox, the command "Point" will be so extended that it can also create a "point on a line" and even produce "intersection points". The following list shows all possible cases:

    • When you click at a free spot of your drawing where there is no line, then a new basis point is created. This is the normal function of the "point" command.
    • When at your click spot there is exactly 1 line, then a new point is created which immediately is bound to this line.
    • When at your click spot there are exactly 2 lines, then this intersection point is created. But if DynaGeo finds more intersection points of the 2 lines, these are created too!
    • When at your click spot there are 3 or more lines, then you'll get an error message, because DynaGeo cannot decide what to do. The command is canceled.

    It is recommended to let this checkbox deactivated. The clear concept of separated commands is better than a superficial "comfort".

Animate the correctness check:

    When you check this box, then the dragging process during a correctness check is dispayed on the screen, so you can look how DynaGeo thinks! But this may be rather time-consuming. If you want to work faster, it is recommended to let the box unchecked. Then the dragging process is only calculated internally.

Use fat mouse pointers:

    This option is provided for persons that are partially sighted. If you check this box, a second mouse cursor set is used with fat cursor pictures that can be seen easier.

XML output format:

    Here you can choose between 4 different internal formats for the GEO files. If you don't have strong reasons to alter the preset, it is best to stick to the recommended choice ("short rows").
