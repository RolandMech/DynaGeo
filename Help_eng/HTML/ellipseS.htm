Ellipse from centre, main vertex and a m.a.c. point    

This command creates an ellipse with a given centre point, a given main vertex point and a minor auxiliary circle that runs through a given third point.

First click the centre point, then click the point that should serve as main vertex, and then click the point the minor auxiliary circle is to run through. (This will set the radius of the "m.a.c.".)

    If the third point lies outside the major auxiliary circle defined by the first two points entered, then the two auxiliary circles change their roles: the first parameter point now defines a minor vertex instead of a (major) vertex.
