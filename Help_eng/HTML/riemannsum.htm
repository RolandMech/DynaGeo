Riemann sum

This commands creates a "Riemann sum", that is a series of slim rectangles (of equal width). Such a Riemann sum is an approximation of the integral area between a function graph and the x-axis. To determine the desired Riemann sum, follow these 3 steps:

  1. Click on the function graph.

  2. Then the interval of integration must fixed: click (with the mouse) on 2 points the x-co-ordinates of which determine the interval. As usual, you can use existing points or create new ones by clicking on an empty spot of your drawing. If the 2 points are free basis points, then they are snapped to the x-axis.

  3. Finally, a window appears, where you select the type of the Riemann sum (Upper or lower sum) and determine in how many subintervals the interval of integration is to be divided. This number normally is from the range [1...256].

Then an appropriate Riemann sum object is created and displayed.


In the context menu of a Riemann sum you find an additional menu item:
  • Edit number of subintervals...
    Calling this command shows a window where you can edit the number of subintervals.



Please note:
  • To limit the time needed for the calcuation of a Riemann sum, the maximal number of subintervals is set to 256. If for special reasons you need more subintervals, then you can edit this value on the register page "Options | Measure".
