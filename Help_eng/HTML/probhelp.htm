How to get further help with more difficult problems

  1. If your EUKLID DynaGeo behaves strangely or has any kind of errors, then first try this help file.

  2. When you cannot find anything helpful there, please look at the EUKLID DynaGeo homepage to see if there can be found anything about your problem.

  3. If not, send me an e-mail with a problem description and perhaps an attached GEO file showing the problem (if several then please ZIP them!).
    The more accurate your bug description, the greater the chances I can help you! Always include some information about your hardware (processor speed, graphic card, memory) and software (operation system version, network software)!
    In any case I shall try to help you as fast as possible.
