Create private menu configurations

A click on the New button in the dialogue Options | Menu configuration opens a dialogue where you can specify a new private menu configuration. You must give it a new unique name. Then please check all the commands that you want to become disabled when this configuration will be applied. The command list only holds those commands that seem suitable for being disabled.

If you close this dialogue by clicking Save, the data of your configuration will be written to your private configuration file EDG_USER.INI. Furthermore, your new menu configuration is added to the list of available menu configurations in the dialogue Options | Menu configuration.
