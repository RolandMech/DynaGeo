Select viewer commands


This command opens a window containing a list of commands. You can choose which of these commands you want to provide when your drawing appears in a HTML document and is presented by the DynaGeoX viewer or the DynaGeoJ viewer. Please mark all commands that you want to appear on the viewer's tool panel. You can choose a maximum of 12 commands. But additionally you are limited by the height of the viewer window: you can only choose as many commands a tool buttons find place in the viewer's tool panel.

When you close the window by a click on the "Okay" button in the upper right corner, the selected commands are stored in your drawing. If you click the "Cancel" button, your changes are discarded.
