Create a new macro

Before creating a new macro you have to carry out the complete construction that later should be constructed by this macro. Then the macro is created in three steps:


  1. Upon selecting the create new macro command, a small window is opened in the working area. The only button of this window is Ok, done!. Now you must click on the start objects of the macro. If a start object required is hidden behind the Ok, done! window, you can move this window to a different position on the screen by dragging it.

    If the selection of a start object is not unique, a window is opened displaying the possible relevant objects. Select the object from the list.

    When all the start objects you need for the macro are selected, click the Ok, done! button.


  2. After you have selected all your start objects the title bar of the Ok, done! button window changes: you are now requested to select the target objects for the macro in the same way you selected the start objects. Click on the Ok, done! button to finish this selection process, too.

    If you select a target object which cannot be derived from the start objects, an error message appears on the screen, and the creation of the macro is canceled.


  3. Finally a bigger dialogue box is opened, in which the name of the macro and a help text may be typed. This name is the menu item that will be appended to the macro menu after the macro is correctly built. The corresponding help text is displayed at any time you ask for help about this macro. This can be done by clicking the F1 key while running the macro.

    The available key commands to format the help text are explained on the page Hot keys.

After you have completed these three steps, the macro is build and added as a new entry to the macro menu.
