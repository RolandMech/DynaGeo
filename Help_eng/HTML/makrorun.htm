Run a macro

All currently loaded macros are displayed in the macro menu. Select a macro, and click on the relevant start object(s). The status bar at the bottom of the EUKLID DynaGeo window always shows the type of the actually expected start object.

After the last start object has been given, the construction steps stored in the macro are carried out and the corresponding objects will be created and displayed on screen.
