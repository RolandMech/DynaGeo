File Menu

The File menu allows you to save, load and print EUKLID DynaGeo drawings. In addition, you may reset the program to a new start, and finally terminate the program when the work is done.
It contains the following items:


    New

    Load
    Save
    Save as
    Save as DynaGeoX HTML page
    Save as DynaGeoJ HTML page

    Printer options
    Print preview
    Print

    Exit

Additionally this menu shows a list of the last recently used files. You can adjust the length of this list under Miscellaneous | Options on the register page Menu configuration.
