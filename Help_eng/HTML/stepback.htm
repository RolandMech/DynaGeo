Undo last construction    

This command takes the construction process one-step-back. The object which was constructed last is deleted, and any snapping of other objects to this one are lost. The Undo last construction command is thus equivalent to deleting the last created object by Edit | Delete object. In both cases the deleted object may be re-activated by the command Edit | Redo last undo.


Please note:

    The command only refers to the last executed construction step. In this way it differs from the one-step-back command in other WINDOWS programs. An example may show the difference: when you
    1. use Construct | Line segment between 2 points to draw a line from point to point, then
    2. use Edit | Edit drawing to colour the end points red, and now
    3. use Edit | Undo last construction
    then the points will not return to black (as when you use the one-step-back in e.g. PaintBrush), but the line will be deleted. The points will remain red.
