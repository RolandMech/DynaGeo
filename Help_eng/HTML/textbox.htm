Create a new textbox

This command displays an editor window in which any desired text may be put. When the window is closed, the text is displayed in your drawing in a rectangular area but without a frame. The size of this textbox is defined during the editing phase. The box is as wide as the editor window from which it receives its text, and high enough to contain the entire text. It may be dragged with the mouse, and so placed at another location.

Additionally, you can bind the textbox to an object of your drawing (see Snap textbox to object). Then it will follow all movements of this object.

The textbox can also be worked on. A double click with the left mouse button in the text displays the editor window again, where the text can be changed or a new width set.


The context menu of a textbox holds the following items:

  • Hide :
      This command allows to hide the textbox the same way as with all the other objects of your drawing.

  • Colour... :
      This brings up a colour dialogue to choose the text's colour. Please note that there is only one colour for the whole of the text.

  • Edit... :
      allows to edit the contents of the text box. This is equivalent to a double click on the textbox.

  • Snap textbox to object... resp. Unsnap textbox from object :
      If the textbox is still unbound, it can be snapped to an object of your drawing. Just indicate the object the textbox is to be bound to. The other command will release a bound textbox and convert it back to a free one.



Please note:

    Textboxes can considerably reduce the program performance in drag mode. Therefore use them with care and only sparingly. (A textbox can theoretically hold up to 10000 characters, but you’d better not try this!) You may however hide textboxes like normal EUKLID DynaGeo objects, and only make them visible when required.
