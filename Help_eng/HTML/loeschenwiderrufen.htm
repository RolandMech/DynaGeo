Redo last undo    

This command revokes your last delete, and re-creates the last deleted object.&xnbsp;It does not matter if you deleted the object by Undo last construction or by Delete object. The Redo last undo command works as if the deleted object is newly created.


Please note:

    Even if you deleted a whole series of objects at one go by your last Delete object command, the Redo last undo command only affects one single object, i.e. the last deleted object. To re-create more objects you have to apply the command repeatedly.
