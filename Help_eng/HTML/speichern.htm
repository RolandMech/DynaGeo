Save    

If the EUKLID DynaGeo title bar already contains a valid file name, then the current drawing is saved under this name. An already existing file using this name will be overwritten without asking for permission.

If the file name in the title bar is NONAME.GEO, a dialogue window is opened to let you specify a valid file name. When you confirm (with Ok) the current drawing is saved to this file, and the title line will display the file name.
