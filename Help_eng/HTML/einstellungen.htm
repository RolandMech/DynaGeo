Options

This command serves to change the options that influence the object display.
Wherever meaningful, a range of valid values is given in the following description, in the format

      [ Minimum.....(Standard value).....Maximum ]

Due to the reasonable number of options they are distributed over several pages of a tabbed notebook:


    StartUp
    Presentation
    Measurements
    Internals
    Menu configuration
    Text settings
    Export
    Mappings
    Locus lines

Below these pages there are two buttons:

  • a Cancel button to leave the window and reject the changes;
  • an Apply these settings button to set all options to the new values.

On the upper border of the window there is a local menu that allows you some special operations concerning the option's management:

    Option data file menu
