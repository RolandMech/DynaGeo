Construct commands in DynaGeoX and DynaGeoJ

Beside purely displaying dynamic drawings the viewers DynaGeoX und DynaGeoJ permit the construction of new objects too. It's you who decides which construction commands will be available to the user of your html page. This is done during exporting the drawing: in the dialog "Save as DynaGeoX HTML page" or "Save as DynaGeoJ HTML page" just click the button "Viewer commands...". A dialog "Select Viewer commands" appears which presents a check list with the available construction commands. Just check the ones you want to be shown to your user. When later the HTML page will be dispayed in a browser, there will be a vertical tool bar at the left side of the viewer window holding a button for every command you selected.

The number of commands you can select is limited to at most 12, and if you use a shareware version, there are no more than 3 commands available. If the number of selected commands has reached its maximum, the rest of the command list items are disabled and you cannot select any further commands. Only when you deselect one of the previously chosen commands, the remaining commands will be reactivated.


Additionally, the viewers provide some more commands in the context menues:

  • Clicking with the right mouse button on an empty spot of the drawing will evoke the context menu of the drawing. It holds commands to enlarge or reduce the drawing, reset it to the original state and write the actual drawing to a file. Additionally, a command to delete any of the objects is provided.

      But please be careful when deleting objects: the viewer cannot undo a delete action! This comfort is only provided in the original DynaGeo program itself.

  • Clicking with the right mouse button on an object of the drawing will evoke the context menu of this object. The menu entries will depend on the type of the object. Usually, you can change the attributes of the object's representation in the drawing, like colour or line style. Furthermore, the context menu of a (free) point allows to snap the point to a line and to unsnap it from the line it is bound to.

      Although the context menu of an object holds an entry to "Hide" this object, you'd better be careful with this command: there is no easy way to recall a hidden object in the viewers, because you cannot open the context menu of a hidden object! In case of emergency you can use the command "Show all hidden objects" from the context menu of the drawing - but this will make visible all the hidden objects of the drawing!


These construction commands considerably expand the didactical scope of the viewers. Nevertheless, the viewers are only meant to allow the students to work with geometrical worksheets. The production of these worksheets is definitely left to the original DynaGeo program. To keep the use of the viewers simple, their construction capabilities will always remain far below the level of the DynaGeo program itself. So if you miss any feature in the viewer while constructing a tricky drawing, please use DynaGeo instead!
