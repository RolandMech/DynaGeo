Convert a point into a free basis point

This command can convert an existing constructed point into a free basis point and preserve all objects that depend on this point. So you can "generalize" your drawing: a previously constructed point now can be freely moved.

After selecting this command, click with the left mouse button on the constructed point. Then it is converted into a free basis point. If the selected point already is a basis point but is snapped to a line, then it is unsnapped from this line.


Attention !

    Please note that usually there is no easy way to undo the execution of this command. Therefore it's a good idea to save your drawing before you apply it. So you can go back to the earlier state if you regret the consequences of your manipulation.
