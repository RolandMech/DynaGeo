Rotate an object    

(automatically repeatable)

Indicate the object to rotate, the centre of rotation and the angle of rotation.

Before you can perform a rotation, its angle must exist in the drawing. There are different possibilities how to provide an angle of rotation:

  • You can use one of the angles that already exist in your drawing. If e.g. one of the angles of a triangle is already marked, you can click on its angle arc. Then the actual width of this angle is used as rotation angle.

  • You can click on an existing number object (or an existing term object). Then its actual value is used as the width of the rotation angle. Please note that this value is supposed to be in radians. If you want the value to be interpreted as degrees, please add a degree sign (°) to the term!
  • You can enter any term that may contain several references to objects in your drawing. The value of the term is also taken to be in radians. If you want it to be in degrees, please add a degree sign (°) to the term. (If it's a sum of several members, you may put brackets around the whole term before adding the degree sign.)

The corresponding rotated object is created and displayed. Each defining point of the original object is connected with its image point by a dotted trace line, as long as you did not deactivate the "automatic trace generation" in Options | Mappings.



Please note:

  • DynaGeo takes the rotation angle always positiv oriented, i.e. it rotates counterclockwise. If you want to rotate clockwise, please provide a negative angle value.

  • If you let this cammand repeat automatically, all subsequent rotations use the same rotation centre and rotation angle as the first one.
