Measure angle    

(automatically repeatable)

Click on an angle’s arc. Then the size of the angle stated in degrees is displayed in a small frame. The angle’s size is never negative.

If you have not yet marked the angle you want to measure, and thus no arc is displayed, then you may identify the angle by 3 defining points in the following order:

  1. Select a point on the first side of the angle
  2. Select the vertex of the angle
  3. Select a point on the other side of the angle

The angle is then marked and measured.

You may drag the angle size display with the mouse.
