Mapping | Define a dilation    

Indicate an original point, the centre of dilation and the image point. A term object is created and displayed, that shows the appropriate factor of dilation.

Please note that the only way to edit the factor of dilation afterwards is to move the defining points. You cannot edit the value of a term object directly. If you want to have full numerical control of the dilation factor, you better create a number object and use its value for subsequent dilations.
