Perpendicular bisector    

(automatically repeatable)

Using the left mouse button, click on a line segment or on two points. A perpendicular bisector is drawn on the line segment between the two points.



ALT variant: Radical axis of 2 circles

    If you start this command by clicking the tool button on the Construct bar, and you hold down the ALT key while you click, then 2 input objects are expected. Each of them can be a circle or a point. Finally the radical axis (or chordale) of these two objects is created and displayed. This line is a generalization of the perpendicular bisector.
