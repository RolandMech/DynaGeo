Create a term object    

A term object shows two lines of text in a frame:

  • in the first line a mathematical expression ("the term") is displayed,
  • the second line shows the term's actual numerical value.

When you choose this command a term input window opens. On the page The syntax of terms you find a list of functions that can be used in terms. After having entered your term, you can close this window by clicking it's Okay button.

Then the term object is created and displayed. It is positioned automatically at a free spot of your drawing. If you want to place it somewhere else, just drag it!


Beside the usual items the context menu of a term object holds another command:

  • Edit term:

    Choosing this command will open up a window where you can edit the term.

    Moreover you can set the display options for the term object:

    • with the "number of digits" you decide how accurate the term value is to be displayed;

    • when you disable the option "Show expression" then the first line of the term object display is omitted and only the value is displayed, but now in a smaller frame.

    If you have selected "Show expression", then there are two more options available:

    • "Show name" decides if in front of the term the internal term name is displayed;

    • "Show comment" allows you to enter a text that will be displayed behind the term. This text is automatically set in curly brackets.
