Edit points and lines   


Calling this command will switch to "Edit mode", show the Shape&Colour bar and activate the first command in this tool bar. Then you can edit the appearance of all point-like and line-like objects of your drawing. This includes object names, measurements, number objects and textboxes, too. Only filled areas are excluded because they have an edit command of their own (Fill area).

To the right of the command button there are three attribute symbols showing the actual values of the object colour, the point style and the line style. If you click on an object in Edit mode the object will adopt these graphic parameter values. If on the other hand you want to change an attribute's value, just click the appropriate button: a local menu appears where you can choose the new value. If you hold down the ALT key while moving the mouse, the cursor will change to a pipette. This allows to take over a clicked object's attribute values into the Shape&Colour bar: the actual attribut values of the Edit mode are set to this object's values and the attribute buttons change their appearance accordingly.

When you are in Edit mode and click on the screen where there are several objects, EUKLID DynaGeo must find out which object you mean. Therefore, the different object types are sorted in an order of decreasing priority:
    Point / Line segment / Line / Circle / Name / Measurement object / Polygon.
If, when applying this priority list, EUKLID DynaGeo is able to distinguish without doubt which object you mean, then this object will be edited. If however, the cursor is in a position where e.g. two different line segments - but no point - might be in question, EUKLID DynaGeo will open a selection box asking you to specify.

The Edit mode will stay active until you terminate it, e.g. by pressing the ESC key or by changing to another command or another toolbar.
