Repeat last macro    

This button is visible only if at least one macro is loaded.

If you click this button the macro addressed last is run. This may be the last recently run macro or the last recently loaded one, depending on which macro command was executed last.

If you delete all macros and the macro menu holds no user defined macro entries, then the Repeat last macro button becomes invisible. It will appear again when you load a macro from a file or create a new one.
