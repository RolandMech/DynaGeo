Basis line    

(automatically repeatable)

Click the mouse at any position in the working area and hold down the mouse button while you move the mouse. Then a line segment is drawn from the start position to the actual mouse spot. Only when you release the mouse button the full basis line is drawn. It will run through the start position and the spot where you released the mouse button.

In contrast to an normal line you cannot afterwards change the direction of a basis line afterwards. You can slide it, but not turn it.
