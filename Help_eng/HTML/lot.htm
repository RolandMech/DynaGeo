Perpendicular

Drop/Raise a perpendicular    

(automatically repeatable)

With the left mouse button, first click on the point where the new perpendicular line should pass through, and then click on the line from/to which the perpendicular line is dropped/raised.

The perpendicular line is created and displayed.



ALT variant: Normal segment from a point to a line

    If you start this command by clicking the tool button on the Construct bar, and you hold down the ALT key while you click, then the same parameter objects are expected, but instead of a perpendicular line only a perpendicular segment is created. This segment starts in the given point, runs orthogonal to the given line and ends on it.
