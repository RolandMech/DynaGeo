Configure correctness check


DynaGeo provides the possibility that teacher can prepare worksheets, give them to the students, let them (try to) solve the posed problem and - finally the student itself can make DynaGeo check if his solution is correct or not. In the window "Configure correctness check" the author of such a worksheet can prepare this automatic correctness check:


  • In the field "condition of correctness" you can enter a Boolean term that is true if and only if the construction is correct. Details about the syntax of Boolean terms can be found on the page Boolean terms. Some examples of correctness conditions are in Basics about the correctness check.

  • In the table below, DynaGeo inserts all of the target variables (@1, @2,....@9) that are used in the condition of correctness. Please select for every of the variables in the list, of which type the correponding target object has to be.

  • In the field "Hint text" you can enter a short help text. This text is shown to the student in the status line when he is indicating the target objects of his construction.

  • The last 2 edit fields may contain "Links" to a next lesson, but separate for the 2 cases of successful and not successful working on the problem. For every case you may enter the desired GEO file that contains an appropriate next worksheet. If all files are in the same folder, the file's name is sufficient. In all other cases, you should provide the relative path from the actual GEO file to the desired next one.
    You can also leave these fields blank, if there are no suitable worksheets (yet).

If you leave this window by clicking on the "Okay" button, then the data entered are written to the drawing. A click on the "Cancel" button will discard your changes.
