Show construction text

This command displays a window in which a construction description of the present drawing is listed. One line of text is given for every object constructed. (This is the same text which is also displayed as an object description during Replay). The description of hidden objects is shown in simple round brackets. If an object is invalid then its description is enclosed in double curly brackets.

The window size and position can be changed. Though the text cannot be edited, it can be marked and then with CTRL-INSERT copied into the clipboard. From there it can be transferred to a word processor for further editing. Additionally, the window offers a search function:

    A double click with the left mouse button on any word of the text will mark the whole line this word is in. Then the object described in this line will start blinking.

    After having marked a line, you can move the marking up and down with the cursor keys (oder the mouse wheel) and thus go through the whole construction, while always the marked object is blinking.


To continue working on the drawing, the construction text window must be closed.
