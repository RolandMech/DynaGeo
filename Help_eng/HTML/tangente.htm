Tangent to a curve    

This command creates a tangent line in a given point to a given curve.

First click the point on the curve that should be the point of tangency. Then click on the curve, if necessary. Maybe, DynaGeo can decide by itself which curve is the one you mean, just because there is only one curve running through your point of tangency. If DynaGeo has learned which point and which curve you mean, then the tangent line is created and displayed.

The point of tangency must exist before you run this command, and it must be a point of the curve. A basis point snapped to the curve is sufficient. Possible curves are circles, arcs, conics, locus lines and function graphs.



ALT variant: Normal line in a point of a curve

    If you start this command by clicking the tool button on the Curves bar, and you hold down the ALT key while you click, then the same parameter objects are expected, but instead of a tangent the corresponding normal line is created.
