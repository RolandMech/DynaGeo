Define an affine transformation...    

Using this command starts an "assistent" that is to help you with the definition of an affine transformation. However, to get this job done successfully some knowledge is needed about this topic.

  • In the first window of the assistent you can select which type of affine transformation you want to build. There are the following items available:

    • Shear
    • Simple orthogonal elongation
    • Shear reflection
    • Affinity with an axis
    • Scaling
    • Affine rotation
    • General affinity through 3 pairs of points
    • General affinity through 6 coefficients (matrix)

    The first 4 transformations of this list are so called affinities with axis, so they all have one straight line that keeps fixed point by point. The next 2 entries hold affinities with exactly 1 fixed point. The last 2 items provide the possibility to define a general affinity, either in the "geometric way" by a triangle and its mapped picture or in the "algebraic way" by providing the coefficients of the mapping's matrix.

  • In the next windows, you are asked for defining objects of the selected transformation. You are guided through this procedure step by step. If necessary, you can select existing objects of your drawing by clicking them with the mouse.

  • In the final window, you are informed about the successful creation of your new affine transformation. Internally, an appropriate "transformation object" is created, and a new item is added to the submenu "Map object by..." of the Mapping menu.


To use the newly created transformation and map an object, just select the new item in the submenu "Map object by...". Then specify the object you want to be mapped. Then the mapped object is created an displayed.

The submenu with the available transformations can also be called with a click on the Map object by... button on the Mappings bar.
