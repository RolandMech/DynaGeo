Measure area    

(automatically repeatable)

Click with the left mouse button on the area you want to measure. You can choose any filled area except angle marks.

Then the actual size of this area is calculated and shown in a little label near the area's centre of gravity. You can change the relative position of this label by dragging it with the mouse.


Remarks:

  • If DynaGeo knows an algorithm to calculate the size of the area, then of course this algorithm is used to get the result. This is the case for polygon and circle areas. But when the border of an area is of more complicated shape (e.g. a combination of several parts of different area borders) then DynaGeo cannot calculate the true value of this area. As a replacement, an estimated value is derived from the actual number of pixels that belong to the representation of the area on the screen.

  • In case of an integral area, DynaGeo calculates the oriented size of the area. You can decide that DynaGeo may calculate the oriented area size of polygons and circle areas, too. Then every area that is negative oriented (i.e. the border runs clockwise around the area) will have a negative area size value. You find the appropriate switch via menu item Miscellaneous | Options, tabsheet Internals.
