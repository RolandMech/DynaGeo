Dynamic Help Window

Create a dynamic help window

This command creates a dynamic help window. This is a textbox with some extended capabilities:

  1. At runtime, first of all only the first paragraph of the whole text appears. By clicking on the small "arrow pointing down" below the last line you make the next paragraph visible. So the text can be made visible piece by piece. If the end of the text is reached, below the last line an "arrow pointing up" is shown: if you click on it, all the paragraphs of the help text except the first one are hidden.

    The end of each paragraph is set at design time by a click on the "End of paragraph" button that inserts a horizontal line in the text.

  2. The text of a dynamic help window can hold links to other GEO files (or HTML files). You can insert such a link by clicking the "Insert a link" button. This opens a window where you can specify the text of the link and its target file. The path to the target file can be given relative to the path of the actual GEO file. If both files are in the same folder, the pure file name is sufficient.
    If the user double-clicks such a link at runtime, the linked GEO file is loaded.


Otherwise, these dynamic help windows act just like normal textboxes. But while you can edit a normal textbox by double-clicking its text, this is not true for dynamic help windows: here the double-click is needed to activate the links in the text! If you want to edit the help windows contents, you can call the editor by the context menu of the help window: just click the menu item "Edit...", and the editor will come up.

If you want to edit an existing link, place the cursor on the link's text, before you click the Insert a link button. Then the link dialog will be initialized with the data of this already existing link.



Attention !
  • When you want to provide several help items in a dynamic help window, then every "logical leaf" of this text must be framed by "end of paragraph" marks. If your text contains only one "end of paragraph" line, this is deleted because no paragraph exists that is "framed" by these lines.

  • If you close the help window editor and the text of your dynamic help window does neither contain a link nor (valid) "end of paragraph" lines, then the help window looses its "dynamic" state and is reset to a "normal" textbox. When you later edit this textbox again, the buttons Insert a link and end of paragraph will have disappeared.
