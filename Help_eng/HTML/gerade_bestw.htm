Line of determined angle

Line of determined angle    

(automatically repeatable)

This menu item allows you to draw a line which crosses a given line or a line segment (called the "reference line") at a given angle. This is accomplished in the following way:

  1. First a small window is opened where you type the size of the angle (in degrees).

  2. Then click on an existing point on the reference line which the new line is not to cross (to identify the reference line).

  3. Next click on an (also already existing) point on the reference line which the new line must cross (to identify the point).

  4. If you typed a positive number as the angle, EUKLID DynaGeo will wait for you to click on that half plane with respect to the reference line where you want the angle line to run through (to identify on which side of the line the angle should be drawn).

The construction of such a line follows the same procedure as the construction of an angle from three points:

  1. a point of the first side of the angle (here the reference line);

  2. the vertex (here the intersection between the reference line and the new line);

  3. a point on the other side of the angle (here a point on the 'angle' side of the reference line).

Instead of a constant angle value you may type a formula or expression. In this case it is impossible to define a third point for the orientation of the angle. The angle will be oriented according to the sign of the current expression.

Beside the standard items the context menu of a line in determined angle shows an entry named

  • angle...,
    that allows to edit the angle's value (or expression) afterwards.


Please note:

    The reference line does not necessarily have to exist as an object, only two points on the (perhaps imaginary) reference line must exist.

    If the reference line does exist, but no useful points are positioned on it, then these will be constructed by mouse clicks (2) and (3). However, they are not yet snapped to the line. You have to do this yourself. Otherwise there is a risk that the points are released next time you move the reference line.



ALT variant: Ray of determined angle

    If you start this command by clicking the tool button on the Construct bar, and you hold down the ALT key while you click, then the same parameter objects are expected, but instead of a line only a ray of determined angle is created.
