The Toolbars

To simplify the access to often used commands EUKLID DynaGeo offers a toolbar. It is displayed below the main menu and shows a series of small tool buttons. When you click a tool button, the corresponding command will be executed as if it was selected from the appropriate menu. In order to include the majority of EUKLID DynaGeo commands without occupying too much valuable screen space there are six different&xnbsp;tool palettes:


    Main Bar

      contains commands from the menus Files and Edit, as well as commands related to locus lines;

    Construct Bar

      contains commands from the menu Construct, which deal with "simple" objects;

    Mappings Bar

      contains commands from the menu Mappings;

    Curves Bar

      contains commands from the menu Construct that are a bit advanced;

    Shape & Colour Bar

      includes commands for editing the drawing. No menu commands directly correspond to the individual icons;

    Measure & Calculate Bar

      includes the commands which relate to co-ordinates, distance values, angle width and area size, and to the calculation of expressions.

    Animation Bar

      finally includes the control commands for animations. This toolbar is only visible if your current drawing contains a valid steering object.



For easy access to these tool palettes, each of them is named on a small menu tab. If you click there, you activate the corresponding toolbar.

If you wish to repeat a certain command, you may activate automatic repeat: just press the Shift key while clicking the icon in question.
