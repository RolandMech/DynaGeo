Options | Locus lines

Plot new traces as smooth curves:

    If this check box is set, any new trace will be drawn as a smooth curve through the recorded locus points. If not, then all of the locus points are shown.
    Normally checked.

Create new locus lines in dynamic mode:

    Check this option to create a dynamic locus line, if ever possible. If you uncheck this option, only a static trace line is created.
    Normally checked.

Convert new locus lines to standard curves (if possible):

    Checking this option results in every new locus line being tested if the locus points lie on a standard curve (a straight line or a circle). Further details see Automatic trace lines.
    Normally unchecked.

Minimum distance of trace points:

    When a trace is produced, a new point can only be joined to the line when it is sufficiently far away from the last registered point. This minimum distance may be defined here (in pixels).
    Range:&xnbsp;&xnbsp; [ 5 ........ (10) ........50 ]
