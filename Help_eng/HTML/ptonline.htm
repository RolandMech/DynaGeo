Point on a line    

(automatically repeatable)

Click the line you want to create a point on. Possible carrier lines are:

  • straight lines (i.e.: lines, rays and segments)
  • circles and arcs
  • polygons
  • locus lines
  • conics
The point is created and snapped to that line.

To release the binding of this point to the carrier line later you can use the menu command Edit | Unsnap point from line or the corresponding command of the point's context menu.
