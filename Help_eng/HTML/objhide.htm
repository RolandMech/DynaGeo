Hide / show object    

(automatically repeatable)

If you select this command, then all of the objects in your drawing are displayed on the screen, even the invisible ones: they are displayed in a grey colour. You may then click on any object:

  • if it was visible before, it will become invisible now;
  • If it was invisible before, it will become visible now.

If you hide the co-ordinate system, all points with fixed co-ordinates will remain fixed and cannot be dragged. If you wish to cancel the fixing of all these points then you must use the command Delete object instead of Hide/show object to hide the co-ordinate system.


ALT variant: Show all hidden objects

    If you start this command by clicking the tool button on the Main bar, and you hold down the ALT key while you click, then all hidden objects become visible.
