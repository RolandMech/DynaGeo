Join 2 points together

With this command you can unify 2 existing points into one point. The command expects two points as input objects, but they are processed rather differently. In the following text, the first point is called the "source point" and the second one the "target point".

To apply this command, click the "source point" with the left mouse button, and then the "target point". After that, the following actions are carried out:

  1. All objects that depend on the "source point" are uncoupled from it and become dependent on the "target point". Clearly spoken: all children of the "source point" are adopted by the "target point".

  2. Finally the "source point" is deleted.


When such an extensive manipulation of the objects' relations are performed, care must be taken to avoid circular references between the objects. This is the case if the target point itself is dependent from the source point. Then DynaGeo shows an appropriate error message and refuses to execute the command. When this happens you can try to repeat this command once more, but now with reversed order of the two points. The probability is high that then the command will work.



Attention !

    Please notice that normally there is no easy way to undo the execution of this command. With "Redo last undo" you can revive the deleted "source point", but the list of its children will remain empty! So it's a good idea to save the drawing before running this command.
