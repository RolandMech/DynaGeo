Edit Copyright note


This command opens a window where you can edit the Copyright text for your actual drawing.


  1. In a first section "Copyright note" you can add a copyright note to your drawings provided you have a licenced version of DynaGeo installed. This copyright note can only be edited by you. For other users, this edit field appears disabled and cannot be edited when your drawing is loaded.

    Attention !
      Your Copyright note ist not a write protection for the contents of your drawing! Every other user may do further work on his copy of your drawing. Only editing of the Copyright note is disabled, and every copy of the file that originally contained your drawing will carry your Copyright note, even if it is saved under another file name.


  2. In a second section "Actual file attributes" appear the file name, the creation date of the file, the date of the last editing and the DynaGeo version with which the file was last edited. Furthermore, the actual file size is shown. These data cannot be edited here - they are shown only for information.


If you close the window by a click on the "Okay" button in the upper right corner, then your Copyright note is written to your drawing. If you take the "Cancel" button, your changes are discarded.
