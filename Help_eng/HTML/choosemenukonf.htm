Length of list of the last recently used files:

    Normally, the File menu shows a list of the last recently used GEO files. Here you can set the number of files in this list. If you enter "0" then no "LRU list" is displayed.
    Range: [0 ........ (4) ....... 9]

If you run EUKLID DynaGeo in a network the following features are only available after the administrator has allowed you to use them:

  • This page of the options window allows to choose among various menu configurations. Select the configuration that suits your needs from the drop-down menu and close the window with Okay. All commands marked as not available in the chosen menu configuration will be deactivated in the menues of the program.

  • If the configurations provided are not sufficient, then you may produce your own additional ones. The button New... leads to a window where you can define your personal menu configuration(s).

  • The button Edit  allows you to edit the actually selected menu configuration if this is one of your private configurations. If one of the five predefined menu configurations is actually selected this button is disabled.

  • The button Delete deletes the last item from the list displayed in the drop down menu provided that it points to one of your private configurations.
