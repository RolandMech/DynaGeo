Point with co-ordinates (x|y)

Point with co-ordinates (x|y)    

(automatically repeatable)

Use this command to construct a co-ordinate point. The initial shape of a co-ordinate point is a hollow square.

Enter the co-ordinates of the point in the window now opened. You are not restricted to constant co-ordinates: expressions can be used, too.

After having entered a valid "x" co-ordinate, then you can use "x" in the expression for the "y" co-ordinate to refer the actual value of the "x" co-ordinate.
This simplifies the construction of points that move on function graphs - though EUKLID DynaGeo was not really meant to draw function graphs, originally!

When you confirm and close the window by clicking OK the co-ordinate point is drawn at the given position.

In addition to the standard items the context menu of a co-ordinate point holds the following commands:

  • Edit co-ordinates:
    allows to edit the co-ordinate expressions.

  • Unfix:
    converts a co-ordinate point to a free basis point.
