About grouping objects

A new drawing contains no groups. With the menu command "View | Gather objects in groups..." you open a submenu that contains at least the menu item "Create a new group". When you activate this command, a dialogue appears where you can give a name to the new group. Additionally, you must define which way the decision about the visibility of the group's objects will be reached. There are 2 sorts of visibility groups:

  • For static groups you decide during the definition of the group whether their objects will be shown or not. This decision may later be revised by an appropriate action of the user. So you decide only about the initial visibility of the group's objects.

  • For dynamic groups you have to create a visibility condition: if the condition is true then the objects area shown, if not, then they are hidden. The condition is a boolean term which may contain comparisons of measures taken from the drawing.
The button "Add / remove objects" closes this dialogue and opens a new window in the upper left corner. This window allows you to select the objects that should be added to the group. If you click on an object, it begins to blink. All blinking objects belong to the group. If you click on one of the blinking obejcts, it stops blinking and is revoked from the group. When you click the window's "Okay"-button, all the blinking objects are put into the group, the window is closed and the visibility of the group's objects is set according to your previously entered decision.


After the creation of the new group, the user interface is updated:

  • To the submenu of "View | Put objects into groups..." a new menu item is added, which shows the name of the new group. A click on this new item opens the above dialogue, which is now initialized with the data of this group.

  • Additionally in the lower right corner of the status bar appears a small square that symbolizes the group. If you created several groups then several such group symbols will show up. If you hold the mouse pointer on such a symbol, the name of the corresponding group is shown - so you can find out which symbol belongs to which group.

A click with the left mouse button on the symbol of a static group toggles its visibility. If the group is visible, then the symbol shows a black square; is the group is switched to "not visible", the symbol shows a white square. Dynamic groups have a gray square symbol. In this case, the mouse click has no effect at all: it's the condition entered before that decides about the visibility.

There is no command to delete a group. If you want to get rid of a previously created group, just remove all objects from the group: empty groups are automatically deleted - after an appropriate warning.

If an object belongs to several groups at the same time, it will be shown if at least one of the groups is set to visible. The composed visibility is calculated by combining the single visibilities with a logical "or".
