Put objects into groups...

This command allows to put several actually visible objects of the drawing together in a group. The members of such a visibility group can all together be hidden or made visible again with one click. You can have several different groups in one drawing, and an object can belong to different groups at the same time, though this is not recommended for the beginning users.

A click on this menu item opens a submenu that lists all groups created before in this drawing. Additionally, there is a menu item "Create a new group". Every click on such a menu item opens a window, where you can edit the properties of the corresponding group: its name and its type (static oder dynamic). For a static group you can choose if the members initially should be visible or hidden. For a dynamic group you enter a Boolean condition which decides about the visibility of the group members. When you close the window by clicking on the button "Add / remove objects", you can add objects to this group or remove them from it.
