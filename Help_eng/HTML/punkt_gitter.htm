Lattice point

After calling this command, click with the mouse on any spot of the drawing window. Then a new lattice point will be created at the location that has integer coordinates and lies nearest to the click spot. When you drag this point with the mouse, it will jump from one spot with integer co-ordinates to the next, even if the co-ordinate system is not visible.

From a technical point of view, lattice points are just normal basis points, with a step width set to "1" when they are created. You can edit this value later by the command "Step width..." from the context menu of the point. If you change the setting to "move most smoothly", then the point will loose its lattice property and become an ordinary basis point.

The same is true when you snap the lattice point to a line: then it will behave just like an ordinary basis point and follow the line smoothly. When you finally unsnap it from the line, the step width will be used again to calculate the new position of the point, and it will jump like before.
