Replay

This menu item allows the course of the construction of the actual drawing to be replayed like a film.

First the EUKLID DynaGeo window is erased, and only the first object displayed. Every click with the left mouse button (or press of the ENTER key) inserts a further object, and the status displays a short description. Valid objects that have been hidden are shown in grey for the duration of the replay. If the drawing contains objects that have become invalid, then these will be bypassed.

As with every other command, the replay can be canceled with the ESC-key. The normal complete drawing is displayed again.
