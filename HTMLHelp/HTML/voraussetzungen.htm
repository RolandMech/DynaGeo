Voraussetzungen

Prinzipiell läuft EUKLID DynaGeo 3 auf jedem Windows-System ab WinXP ( XP / Vista / Win7 / Win8 / Win10 ... ). Das Programm ist eine 32-Bit-Windows-Anwendung, die sich jedoch auch auf 64-Bit-Windows-Varianten betreiben lässt. Wenn Sie die empfohlenen Hardware-Anforderungen des Betriebssystems erfüllt haben, sollte auch EUKLID DynaGeo laufen.

Das Programm belegt auf Ihrer Festplatte (mit allen Beispieldateien) weniger als 10 MByte. Wenn Sie auf einen ruckfreien Zugmodus auch bei größeren Zeichnungen Wert legen, ist eine flotte Grafikkarte von Vorteil. Obwohl EUKLID DynaGeo im Prinzip keinen speziellen Grafikmodus verlangt, wird zur sauberen Darstellung der Programmoberfläche eine Bildschirmauflösung von mindestens 1024*768 Punkten empfohlen. Bei geringeren Auflösungen kann es zu Gedränge in den Werkzeugleisten kommen.

EUKLID DynaGeo läuft auch in allen gängigen Netzen; speziell für Windows-Netze liegen umfangreiche Betriebserfahrungen vor, die das Programm als genügsam und verträglich ausweisen. Aber auch in Netzen mit Linux-Servern kann DynaGeo betrieben werden, solange auf den Clients ein passendes Windows läuft. Und selbst auf Linux-Clients ist das Programm schon zum Laufen gebracht worden, allerdings nur unter Zuhilfenahme einer Windows-Laufzeit-Umgebung: "WINE" macht's möglich!

Um unnützen Anfragen zuvorzukommen:

  • Nein, DynaGeo läuft nicht auf Apple-Computern - es sei denn, Sie installieren Windows darauf ;-)
  • Nein, es ist weder eine MAC-Version noch eine direkt unter Linux lauffähige DynaGeo-Variante geplant!
