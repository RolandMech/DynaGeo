Punkt auf Gitterpunkt schieben    

(automatisch wiederholbar)

Dieser Befehl schiebt einen (Basis- oder Koordianten-) Punkt auf den nächstgelegenen Gitterpunkt und fixiert ihn dort.

Wenn Sie diesen Befehl auf einen Basispunkt anwenden, wird er automatisch in einen Koordinatenpunkt umgewandelt.
