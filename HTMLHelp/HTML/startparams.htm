Startparameter

Startparameter beim Programmaufruf


a) Grundsätzliches

Windows kann einem Programm beim Aufruf eine Reihe von sogenannten "Kommandozeilen-Parametern" übergeben. Dies sind kurze Text-Kommandos, die - durch Leerzeichen getrennt - dem Dateinamen nachgestellt werden. Dazu müssen Sie das Programm über die Kommandozeile des Startmenüs (Menüpunkt Start | Ausführen) aufrufen: dann zeigt Windows Ihnen ein einzeiliges Editierfeld, in das Sie den vollständigen Pfad zur EXE-Datei sowie die gewünschten Parameter eingeben können.
    Alternativ können Sie auch eine Verknüpfung zu diesem Programm editieren (mit "Eigenschaften" im Kontextmenü der Verknüpfung) und dort dem Pfad zur EXE-Datei die Parameter nachstellen. Beachten Sie aber, dass das Programm dann über diese Verknüpfung immer mit diesen Parametern aufgerufen wird! Für eine einmalige Anwendung eines Start-Parameters ist die oben beschriebene Methode über die Kommandozeile des Startmenüs vorzuziehen.


b) Mögliche Parameter für EUKLID DynaGeo

EUKLID DynaGeo versteht die folgenden Kommandozeilen-Parameter:
  1. den Namen bzw. den vollständigen Pfad einer GEO-Datei, die dann beim Start von EUKLID DynaGeo automatisch geöffnet wird; diese Eigenschaft hat zur Folge, dass durch einen Doppelklick auf eine GEO-Datei im Explorer automatisch EUKLID DynaGeo gestartet und diese Datei geladen wird, sofern im Betriebssystem eine ordnungsgemäße Verknüpfung der GEO-Dateien mit dem EUKLID DynaGeo-Programm registriert wurde;

  2. /debug
    Diesen Parameter sollten Sie nur in sehr speziellen Fällen hartnäckiger Schwierigkeiten mit EUKLID DynaGeo einsetzen. Falls Sie dies tun, startet EUKLID DynaGeo mit einer Diagnose Ihres Windows-Systems. Die Ergebnisse werden in eine Textdatei namens "debug.txt" ins Verzeichnis "{Anwendungsdaten}\DynaGeo" geschrieben, das sich im Profil des aktuellen Benutzers befindet.

      Derzeit steht in dieser Debug-Datei lediglich, welche Windows-Version Sie benutzen und ob bzw. in welcher Version der Internet-Explorer und die HTML-Hilfe-Funktionen auf Ihrem Rechner installiert sind. Außerdem enthält sie ein Protokoll des Startvorgangs von DynaGeo. Darüberhinaus werden in einigen wenigen kniffligen Programmsituationen zusätzliche Laufzeit-Informationen in diese Datei geschrieben. In Zukunft könnte dieser Mechanismus verstärkt auch für gründlichere Diagnosen eingesetzt werden.
