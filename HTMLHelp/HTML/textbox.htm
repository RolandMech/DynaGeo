Textbox einfügen    

Dieser Befehl bringt ein Editier-Fenster zur Anzeige, in das Sie einen beliebigen Text eingeben können. Während der Eingabe stehen Ihnen die üblichen Formatierungsoptionen zur Verfügung: Sie können Schriftart und Schriftgröße wählen, den Text fett, kursiv und unterstrichen darstellen, und selbst tief- und hochgestellte Zeichen sind möglich. Der Knopf "Sonderzeichen" hilft beim Einfügen von häufig gebrauchten griechischen Buchstaben und mathematischen Symbolen. Die Abmessungen der Textbox werden während des Editierens festgelegt: die Textbox wird so groß, dass sie den gesamten im Editierfenster stehenden Text fassen kann.

Nach dem Schließen des Editierfensters mit einem Klick auf Okay wird der eingegebene Text in Ihrer Zeichnung angezeigt. Sie können seine Position in Ihrer Zeichnung ändern, indem Sie ihn mit der Maus verziehen. Darüberhinaus kann eine Textbox auch an ein Objekt gebunden werden, so dass sie dann allen Bewegungen dieses Objekts folgt.

Eine Textbox kann auch nachbearbeitet werden: ein Doppelklick mit der linken Maustaste auf den Text bringt wieder das Editierfenster zur Anzeige. Sie können den Text dann ändern oder die Breite neu einstellen. Achten Sie jedoch darauf, dass beim Schließen des Editierfensters der gesamte Text Ihrer Textbox zu sehen ist. Nur dann ist gewährleistet, dass er auch in der Zeichnung vollständig angezeigt werden kann.


Im Kontextmenü einer Textbox finden Sie die folgenden Einträge:

  • Verbergen :
      Mit diesem Befehl können Sie eine Textbox genau so verbergen, wie jedes andere Objekt Ihrer Zeichnung auch.

  • Farbe... :
      Hiermit rufen Sie den Farbauswahl-Dialog auf, mit dessen Hilfe Sie die Farbe des Textes ändern können. Beachten Sie, dass es für den gesamten Text nur eine Farbe gibt.

  • Editieren... :
      Dieser Befehl erlaubt Ihnen nachträgliche Änderungen des Textes. Er hat dieselbe Wirkung wie ein Doppelklick auf Ihren Text: es wird das Text-Eingabefenster aufgerufen und der aktuelle Text ins Editierfeld geladen.

  • Textbox an Objekt binden... bzw. Bindung der Textbox aufheben :
      Ist die Textbox noch frei, dann kann sie mit Textbox an Objekt binden... an ein beliebiges geometrisches Objekt Ihrer Zeichnung gebunden werden. Sie müssen dazu nur (mit der Maus) das Objekt anklicken, an das der Text gebunden werden soll. Eine schon gebundene Textbox können Sie mit Bindung der Textbox aufheben wieder in eine "freie" zurückverwandeln.


Achtung !

    Textboxen können die Performance des Programms im Zugmodus deutlich verschlechtern. Setzen Sie sie daher sparsam und gezielt ein, und machen Sie sie nicht größer als nötig. (Eine Textbox kann zwar theoretisch sehr viele Zeichen enthalten, aber Sie sollten das nicht ausnützen.)
