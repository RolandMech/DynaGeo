Punkt an Objekt binden

Punkt an Linie binden    

(automatisch wiederholbar)

Klicken Sie zuerst mit der linken Maustaste den zu bindenden Punkt an und danach die Trägerlinie, an die der Punkt gebunden werden soll. Als Trägerlinie können Sie folgende Objekte verwenden:

  • Gerade Linien (also: Geraden, Halbgeraden und Strecken)
  • Kreise (auch: Kreisbögen)
  • Polygone
  • Ortslinien

Falls der Punkt nicht schon vorher auf dem Trägerobjekt lag, wird er sich nun zu ihm hin bewegen, wobei alle von ihm abhängigen Objekte (wie üblich) entsprechend mitgezogen werden.

Wenn Sie im folgenden an diesem Punkt ziehen, dann wird er sich nur noch auf seinem Trägerobjekt bewegen. Er kann es erst wieder verlassen, wenn Sie die Bindung aufgehoben haben.


Achtung !

    Nur Basispunkte können gebunden werden, da ja nur Basispunkte gezogen werden können. Ein Punkt kann jedoch nie an ein Objekt gebunden werden, das mit seiner Hilfe konstruiert wurde (und daher von ihm abhängig ist).
