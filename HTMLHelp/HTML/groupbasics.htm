Grundsätzliches zum Gruppieren

Eine neu erstellte Zeichnung enthält zunächst noch keine Gruppen. Über den Menü-Befehl "Ansicht | Objekte gruppieren..." öffnen Sie ein Untermenü, das zunächst den einzigen Menüpunkt "Neue Gruppe einrichten" enthält. Wenn Sie diesen Befehl anwählen, erscheint ein Fenster, in dem Sie der neuen Gruppe zunächst einen Namen geben müssen. Sodann ist noch festzulegen, wie über die Sichtbarkeit der Objekte in der neuen Gruppe entschieden werden soll. Es gibt zwei verschiedene Sorten von Gruppen:

  • Für statische Gruppen legen Sie bei der Erstellung der Gruppe fest, ob sie angezeigt werden oder nicht. Diese Entscheidung kann später jederzeit revidiert werden, aber stets nur durch eine entsprechende Aktion des Benutzers. Sie legen also zunächst nur den Anfangswert für die Sichtbarkeit fest.

  • Für dynamische Gruppen geben Sie eine Bedingung an, anhand welcher über die Sichtbarkeit der Gruppenobjekte entschieden wird: ist die Bedingung erfüllt, werden die Objekte angezeigt, ist sie nicht erfüllt, werden sie verborgen. Diese Bedingung ist ein boole'scher Ausdruck, in dem auch aus der Zeichnung entnommene Größen (also Abstands-, Winkel-, Zahl- und Termwerte) verglichen werden können.
Der Knopf "Objekte hinzufügen / entfernen" schließt dieses Fenster und öffnet links oben ein neues, welches Ihnen erlaubt, von den Objekten der Zeichnung diejenigen auszuwählen, die in diese Gruppe eingefügt werden sollen. Wenn Sie in diesem Zustand ein Objekt anklicken, wird es angewählt und beginnt zu blinken. Wenn Sie auf ein blinkendes Objekt klicken, dann hört es auf zu blinken und ist damit wieder abgewählt. Wenn Sie auf den "Okay"-Knopf des Fensters klicken, werden alle derzeit blinkenden Objekte in die Gruppe aufgenommen, das Fenster geschlossen und die Sichtbarkeit der gruppierten Objekte entsprechend der von Ihnen festgelegten Vorgaben gesetzt.


Nach dem Einrichten einer neuen Gruppe wird die Benutzer-Oberfläche an die neuen Verhältnisse angepasst:

  • Dem Untermenü von "Bearbeiten | Objekte gruppieren..." wird ein neuer Eintrag hinzugefügt, der den Namen der Gruppe zeigt. Ein Klick auf diesen Menüpunkt öffnet wieder das erste der oben beschriebenen Fenster, welches aber in diesem Fall mit den Daten der schon existierenden Gruppe initialisiert wird.

  • Außerdem erscheint rechts unten in der Statuszeile ein kleines Quadrat, das die Gruppe symbolisiert. Wenn Sie mehrere Gruppen eingerichtet haben, werden entsprechend mehrere solche Gruppensymbole nebeneinander aufgereiht. Wenn Sie mit der Maus über einem solchen Gruppensymbol verharren, wird der Name der zugehörigen Gruppe angezeigt - so können Sie herausfinden, welches Symbol zu welcher Gruppe gehört.

Ein Klick mit der linken Maustaste auf das Symbol einer statischen Gruppe schaltet deren Sichtbarkeit um. Ist die Gruppe sichtbar, dann zeigt das Symbol ein schwarzes Quadrat; ist die Gruppe auf "nicht sichtbar" geschaltet, zeigt das Symbol ein weißes Quadrat. Dynamische Gruppen werden durch ein graues Symbol dargestellt. In diesem Fall bleibt ein Mausklick auf das Symbol ohne Wirkung: über die Sichtbarkeit wird anhand der eingegebenen Bedingung entschieden.

Es gibt keinen Befehl, um eine Gruppe zu löschen. Wenn Sie eine einmal eingerichtete Gruppe wieder loswerden wollen, müssen Sie einfach alle Objekte aus ihr entfernen. Leere Gruppen werden (nach einer entsprechenden Rückfrage) automatisch gelöscht.

Wenn ein Objekt gleichzeitig zu mehreren Sichtbarkeitsgruppen gehört, so wird es genau dann angezeigt, wenn mindestens eine der Gruppen auf "sichtbar" geschaltet ist. Über die Sichtbarkeit eines Objekts wird also entschieden, indem die Sichtbarkeits-Attribute aller Gruppen, zu denen dieses Objekt gehört, mit "oder" verbunden werden.
