Vorhandenes Makro löschen

Sind im Augenblick mehrere Makros verfügbar, wird Ihnen eine Liste dieser Makros gezeigt. Wählen Sie das zu löschende Makro aus. Es wird sodann aus dem Arbeitsspeicher entfernt und der entsprechende Menüeintrag im Makro-Menü gelöscht.


Achtung !

    Falls es eine zugehörige Makrodatei gibt, dann wird diese durch diesen Befehl nicht gelöscht. Falls Sie Makrodateien löschen wollen, können Sie dies z.B. mit dem Datei-Explorer von Windows tun. Makrodateien tragen die Standarderweiterung ".MAK".
