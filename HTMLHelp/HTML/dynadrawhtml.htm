Dynamische Zeichnungen in HTML-Dokumenten

EUKLID DynaGeo erlaubt die Darstellung von dynamischen Zeichnungen, die mit EUKLID DynaGeo ab Version 2.4 erstellt wurden, in HTML-Dokumenten. Damit können Sie also auch in einem HTML-Dokument eine Zeichnung zur Verfügung stellen, die der Leser Ihrer Seite mit der Maus verziehen kann. Dies kann auf zwei verschiedenen Wegen erreicht werden:

  • Sie verwenden den mit DynaGeo (ab Version 2.4) gelieferten DynaGeoX-Viewer, der die ActiveX-Technologie von Microsoft benutzt.

  • Sie verwenden den mit DynaGeo (ab Version 3.5) gelieferten DynaGeoJ-Viewer, der eine JAVA-Laufzeitumgebung auf Ihrem Rechner voraussetzt.

Wenn Sie DynaGeoX einsetzen wollen, sollten Sie beachten, dass auf jedem Rechner, auf dem Ihre HTML-Dokumente dargestellt werden sollen, mindestens Windows 98 laufen sollte und als Browser zwingend der Microsoft Internet Explorer (ab Version 5) benutzt werden muss.
