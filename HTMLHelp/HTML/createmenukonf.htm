Eigene Menükonfigurationen erstellen

Ein Klick auf den Schalter Neu... im Dialogfenster Einstellungen / Menükonfiguration öffnet einen Dialog, in dem Sie eine eigene Menükonfiguration spezifizieren können. Sie müssen dazu dieser Konfiguration einen eigenen eindeutigen Namen geben und dann in der Liste der deaktivierbaren Befehle diejenigen markieren, die in dieser Konfiguration nicht zur Verfügung stehen sollen. Die Liste enthält nur diejenigen der möglichen Befehle, für die eine Deaktivierung sinnvoll erscheint. Wenn Sie den Dialog mit einem Klick auf Speichern verlassen, werden die Daten ihrer Konfiguration in Ihre private Konfigurationsdatei EDG_USER.INI eingetragen.

    In früheren Versionen von EUKLID DynaGeo befand sich unter den deaktivierbaren Befehlen auch der Befehl Einstellungen ändern, was zu sogenannten "verriegelten Konfigurationen" führte: ein Benutzer konnte sich damit den Zugang zu den Einstellungen abschneiden und sich damit selbst aussperren. Dies ist nun nicht mehr möglich, weil der Befehl Einstellungen ändern aus der Liste der deaktivierbaren Befehle herausgenommen wurde. Soll der Befehl Einstellungen nun deaktiviert werden, dann kann dies über die Netzwerk-Einstellungen erreicht werden.
