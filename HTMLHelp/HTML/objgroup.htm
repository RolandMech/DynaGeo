Objekte gruppieren

Dieser Befehl erlaubt es, beliebige sichtbare Objekte der Zeichnung in Gruppen zusammen zu fassen, so dass sie dann gemeinsam verborgen bzw. wieder angezeigt werden können.

Ein Klick auf diesen Menüpunkt öffnet ein Untermenü, in dem alle bisher eingerichteten Gruppen aufgelistet sind. Zusätzlich ist noch der Befehl "Neue Gruppe einrichten" hinzugefügt. Jeder Klick auf einen Eintrag dieses Untermenüs öffnet ein Fenster, in dem die Kenndaten der entsprechenden Gruppe editiert werden können: der Gruppenname und die Art der Gruppe (statisch oder dynamisch). Für statische Gruppen können Sie den Startwert für das Sichtbarkeits-Attribut einstellen, für dynamische ist eine boole'sche Bedingung einzugeben, deren Wert über die Sichtbarkeit entscheidet. Wenn Sie das Fenster mit "Objekte hinzufügen / entfernen" schließen, können Sie Objekte der Zeichnung zu dieser Gruppe hinzufügen bzw. aus ihr entfernen.
