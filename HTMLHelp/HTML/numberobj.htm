Zahlobjekt erstellen    

Ein Zahl-Objekt ist ein Schieberegler, mit dessen Hilfe Sie eine variable Zahl aus einem von Ihnen vorgegebenen Intervall zur Verfügung stellen können, um sie in Ihrer Zeichnung weiterverwenden zu können. Mit Zahlobjekten lässt sich z.B. ein variabler Streckfaktor für eine zentrische Streckung zur Verfügung stellen, oder ein Parameter, mit dessen Hilfe Sie einen Punkt mit bestimmten Koordinaten auf einer festgelegten Bahn entlanglaufen lassen können.

Wenn Sie diesen Befehl anwählen, wird ein Zahlobjekt mit Standard-Daten erzeugt und automatisch an einer freien Stelle Ihrer Zeichnung angezeigt. In der oberen Hälfte des Objekts werden die untere Intervallgrenze, der aktuelle Zahlwert und die obere Intervallgrenze angezeigt. Darunter erscheint der eigentliche Schieberegler zur Veränderung des aktuellen Wertes.

    (In früheren Versionen von EUKLID DynaGeo mussten Sie zusätzlich noch angeben, wo in Ihrer Zeichnung das Zahlobjekt dargestellt werden sollte; dies ist inzwischen überflüssig. Wenn Sie mit der automatischen Plazierung nicht zufrieden sind, können Sie das Zahlobjekt ja jederzeit an eine andere Stelle schieben!)

Der Zahlbereich ist zunächst auf das Intervall [-3; 5] festgelegt. Sie können die Grenzen dieses Bereichs über das Kontextmenü editieren (siehe unten).

In der Mitte wird der aktuelle Wert des Zahlobjekts angezeigt; er ist auf 1 voreingestellt. Sie können diesen Wert ändern, indem Sie den Schieber in der unteren Hälfte des Zahlobjekts horizontal verziehen. Der jeweils aktuelle Wert eines Zahlobjekts Z1 kann auch in einem Termobjekt verwendet werden: fügen Sie dazu einfach den Namen "Z1" des Zahlobjekts in den Term ein. Weitere Details zu diesem Thema finden Sie auf der Seite über die Syntax von Termen.


Das Kontext-Menü eines Zahlobjekts enthält zusätzlich zu den üblichen Einträgen noch die folgenden Menüpunkte:

  • Bereich editieren:
    Wenn Sie diesen Menüpunkt anklicken, öffnet sich ein Fenster, in dem Sie die Werte für die Intervallgrenzen und den "aktuellen Wert" direkt setzen können.

    Darüberhinaus können Sie ein "Schrittweite" eingeben: damit können Sie die Werte des Zahlobjekts in ein bestimmtes Raster zwingen, sie sozusagen "quantisieren". Wenn Sie beispielsweise die Schrittweite auf "1" stellen, dann kann das Zahlobjekt nur noch ganzzahlige Werte annehmen; bei Schrittweite "0,2" ist der Wert stets ein Vielfaches von 0,2.

  • Minimiert zeigen:
    Ein Klick auf diesen Menüpunkt schaltet das Zahlobjekt in den "minimierten" Zustand: statt eines großen Schiebereglers wird dann nur noch ein kleines Rechteck gezeigt, das neben dem aktuellen Wert nur noch zwei Pfeile enthält. Jeder Klick auf den oberen dieser Pfeile erhöht den Wert des Zahlobjekts um die jeweils eingestellte Schrittweite, ein Klick auf den unteren Pfeil vermindert den Wert entsprechend.

    Diese Darstellung eignet sich besonders für Fälle, in denen der Bildschirmplatz sehr knapp ist. Dann kann ein minimiertes Zahlobjekt z.B. als Schalter zwischen mehreren verschiedenen Zuständen Ihrer Zeichnung dienen. Eine wesentliche Voraussetzung für den erfolgreichen Einsatz ist aber, dass Sie eine geeignete Schrittweite einstellen, bevor Sie zur minimierten Darstellung wechseln.

  • In Normalansicht zeigen:
    Dieser Eintrag erscheint nur im Kontextmenü des minimierten Zahlobjekts, und zwar statt des oben besprochenen Befehls "Minimiert zeigen". Ein Klick auf "In Normalansicht zeigen" schaltet das Zahlobjekt in den normalen Zustand, also in die Schieberegler-Ansicht zurück.
