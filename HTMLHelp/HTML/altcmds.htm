Verzeichnis der ALT-Befehle


Einige EUKLID DynaGeo - Befehle haben zwar keine eigenen Schaltflächen in den Werkzeugleisten, sind aber doch über die Werkzeugleiste erreichbar, indem beim Klicken auf eine Befehlsschaltfläche die ALT-Taste gedrückt gehalten wird. Bitte beachten Sie, dass diese "ALT-Befehle" grundsätzlich nicht automatisch wiederholbar sind, auch wenn der Hauptbefehl des Werkzeugknopfes diese Eigenschaft hat. Es nützt also nichts, wenn Sie gleichzeitig mit der "ALT"-Taste die "Shift"-Taste gedrückt halten!

Hier werden nun die verfügbaren "alternativen" Befehle aufgelistet:


auf der "Hauptleiste":

  • ALT + :     Alle verborgenen Objekte anzeigen



auf der "Konstruktieren - Leiste":

  • ALT + :     Gitterpunkt


  • ALT + :     Lotstrecke von einem Punkt auf eine Gerade


  • ALT + :     Kreislinie durch 3 Punkte


  • ALT + :     Kreis mit Streckenlänge als Radius ("In den Zirkel nehmen")


  • ALT + :     Chordale zweier Kreise


  • ALT + :     Außenwinkel-Halbierende


  • ALT + :     Halbgerade in bestimmtem Winkel


  • ALT + :     Reguläres N-Eck



auf der "Kurven-Leiste":

  • ALT + :     Riemannsumme für die Fläche unter einer Kurve


  • ALT + :     Normale in einem Punkt einer Kurve



auf der "Messen & Rechnen - Leiste":

  • ALT + :     Streckenlänge messen
