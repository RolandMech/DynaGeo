Geodreieck anzeigen / verbergen

Dieser Befehl bringt ein transparentes Geodreick zur Anzeige, das auf Ihrer Zeichnung zu liegen scheint. Beim nächsten Aufruf des Befehls wird das Geodreieck wieder verborgen.

Sie können das Geodreieck wie ein gewöhnliches Basisobjekt mit der Maus manipulieren. Wenn Sie es im Inneren des Winkelmaß-Halbkreises anklicken, können Sie es verschieben. Wenn Sie das Geodreieck außerhalb dieser Kreislinie an einer seiner Ecken packen, können Sie es um seinen Nullpunkt (= Mittelpunkt der längsten Seite) drehen. Speziell auf langsameren Computern kann es nötig sein, diese Bewegungen vorsichtig und nicht zu schnell durchzuführen.



Bitte beachten Sie:

  • Das Geodreieck ist nur verfügbar, wenn Ihre Zeichnung ein isotropes Koordinatensystem verwendet; bei unterschiedlichen Einheiten auf den Achsen wird es automatisch ausgeblendet.

  • Die Anzeige des Geodreiecks ist eine mühsame Aufgabe für Windows, welche einiges an Systemressourcen braucht. Sie sollten das Geodreieck daher nur so lange einblenden, wie Sie es wirklich brauchen, und es danach gleich wieder verbergen.

  • Man kann mit dem Geodreieck auch Winkel messen. Die Details dazu finden Sie in der Hilfe zum Spiel "Winkel messen".
