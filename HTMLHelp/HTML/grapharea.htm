Fläche unter einem Funktions-Schaubild

Fläche unter einem Funktions-Schaubild    

Dieser Befehl zeichnet ein "krummliniges Trapez", also eine Integralfläche.

    In der folgenden Beschreibung wird davon ausgegangen, dass die "obere" Randfunktion vollständig oberhalb einer eventuell vorhandenen "unteren" Randfunktion bzw. der x-Achse verläuft. Natürlich kann DynaGeo auch Integralflächen zeichnen, für die diese Voraussetzungen nicht gelten. Dass sie hier trotzdem zunächst als erfüllt angesehen werden, dient lediglich der Vereinfachung der folgenden Beschreibung.
Um die gewünschte Fläche festzulegen, führen Sie bitte die folgenden drei Schritte aus:

  1. Geben Sie zunächst das Funktions-Schaubild an, das die Fläche nach oben begrenzt.

  2. Markieren Sie dann das Integrations-Intervall: geben Sie dazu zwei Punkte an, deren x-Koordinaten das Integrations-Intervall bestimmen. Sie können entweder schon vorhandene Punkte benutzen oder neue Basispunkte durch einen Mausklick erzeugen. Sind die Begrenzungspunkte des Integrations-Intervalls ungebundene Basispunkte, dann werden sie bei der Erzeugung der Fläche an die x-Achse gebunden.

  3. Schließlich ist noch die untere Begrenzung des krummlinigen Trapezes festzulegen. Dazu haben Sie zwei Möglichkeiten:

    1. Eine einfache "Fläche unter einer Kurve" wird nach unten durch die x-Achse begrenzt.
      Klicken Sie in diesem Fall auf die x-Achse.

    2. Wenn Sie stattdessen eine "Fläche zwischen 2 Kurven" darstellen wollen,
      klicken Sie auf ein zweites Funktions-Schaubild.
Danach wird ein entsprechendes Integralflächen-Objekt erzeugt und auf dem Bildschirm angezeigt.


Bemerkungen:
  • Die rechte und linke Begrenzung der Fläche wird durch Strecken orthogonal zur x-Achse geliefert. Die obere Begrenzung muss immer durch ein Funktions-Schaubild geliefert werden. Die untere Begrenzung ist entweder durch die x-Achse gegeben, oder durch ein zweites Funktions-Schaubild.

  • Um den Flächeninhalts-Wert einer solchen Fläche in einem Term zu verwenden, müssen Sie die Standard-Funktion area() benutzen. Heißt das Flächenobjekt z.B. "Int1", dann liefert der Term "area(Int1)" den aktuellen Inhalt dieser Fläche. Der Wert gibt jedoch den sogenannten "orientierten Flächeninhalt" an, also das bestimmte Integral: so hat z.B. die Fläche, die eine volle Periode der Sinuskurve mit der x-Achse einschließt, den orientierten Inhalt 0 !

  • Problemlos funktioniert die Berechnung der Integralflächen bei auf ganz R stetigen Randfunktionen. Hat die Randfunktion hingegen einen eingeschränkten Definitionsbereich, dann muss das Intergrations-Intervall komplett innerhalb des Definitionsbereichs liegen. Andernfalls wird die Integralfläche ungültig und verschwindet vom Bildschirm, oder die "area()"-Funktion liefert falsche Werte. Z.B. darf bei gebrochen-rationalen Randfunktionen das Integrations-Intervall keine Polstelle der Randfunktion enthalten.

  • Einen Sonderfall stellen unstetige Randfunktionen mit endlichen Sprungstellen dar, wie z.B. floor(). DynaGeo kann solche Integrale nur dann korrekt berechnen, wenn die Ränder des Integrations-Intervalls beide im sichtbaren Fenster liegen. Die Ergebnisse dieser Berechnungen sind jedoch deutlich ungenauer als die bei stetigen Randfunktionen erzielten.


ALT-Variante: Riemann-Summen

    Wenn Sie den Befehl Fläche unter einem Funktions-Schaubild über die Werkzeugleiste anwählen und beim Klicken auf das entsprechende Icon die ALT-Taste gedrückt halten, dann wird statt der Integralfläche eine Riemann-Summe erzeugt. Es sind dazu einige zusätzliche Eingaben nötig, deren genaue Beschreibung Sie in der Hilfe zum Befehl Riemann-Summen finden.
