Tangente an eine Kurve

Tangente in einem Punkt an eine Kurve    

Dieser Befehl zeichnet die Tangente in einem gegebenen Berührpunkt an eine Kurve.

Geben Sie zunächst den Punkt auf der Kurve an, in dem die Tangente die Kurve berühren soll. Falls EUKLID DynaGeo nicht schon jetzt erkennen kann, welche Kurve Sie meinen, erscheint in der Statuszeile (am unteren Fensterrand) ein Hinweis, dass Sie nun noch die Kurve anklicken müssen. Danach wird das entsprechende Tangenten-Objekt erzeugt und die Tangente angezeigt.

Der eingegebene Punkt muss schon existieren, und er muss auf der zu berührenden Kurve liegen. Sie können dies zum Beispiel dadurch erreichen, dass Sie vor der Anwendung des "Tangenten"-Befehls einen Basispunkt erzeugen und an die Kurve binden. Als mögliche Kurven kommen nahezu alle gebogenen Linien einer DynaGeo-Zeichnung in Frage: Kreise und Kreisbögen, Kegelschnitte, Ortslinien und Funktions-Schaubilder.

Speziell bei Tangenten an Funktions-Schaubilder ist es häufig wünschenswert, dass man die x-Koordiante des Berührpunktes der Tangente nachträglich genau justieren kann. Zu diesem Zweck finden Sie im Kontextmenü eines (an einen Funktions-Graphen(!)) gebundenen Punktes einen zusätzlichen Eintrag:

  • x-Koordinate editieren...
    Wenn Sie diesen Menüpunkt anklicken, erscheint ein Fenster, das mit dem aktuellen Wert der x-Koordinate des gebundenen Punktes initialisiert wird. Geben Sie dann den gewünschten neuen Wert ein. Ein Klick auf "OK" schließt das Fenster und schiebt den Punkt auf dem Funktions-Graphen an die gewünschte Stelle.


ALT-Variante: Normale in einem Punkt einer Kurve

    Wenn Sie den Befehl Tangente an eine Kurve über die Werkzeugleiste anwählen und beim Klicken auf das entsprechende Icon die ALT-Taste gedrückt halten, dann erwartet der Befehl genau dieselben Eingabeobjekte, erstellt dann aber statt der Tangente die Normale auf die Kurve in dem gegebenen Kurvenpunkt.
