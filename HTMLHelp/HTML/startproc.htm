Initialisierungen beim Programmstart

Wenn Sie EUKLID DynaGeo starten, erscheint zunächst ein Titelfenster. Dieses zeigt neben dem Programmnamen, der Versionsnummer, einem Copyrightvermerk und dem Programmsymbol die Lizenzdaten des vorliegenden Exemplars von EUKLID DynaGeo an: bei einer lizenzierten Version sind dies Name und Anschrift des Lizenznehmers sowie die Lizenznummer; bei einer Freeware-Version erscheint lediglich der Hinweis, dass es eben eine solche ist.

Am unteren Rand des Titelfensters erscheint ein Schalter, der zunächst die Aufschrift Bitte warten trägt. Solange Sie diese Aufschrift sehen, hat ein Anklicken dieses Schalters keine Wirkung. EUKLID DynaGeo braucht etwas Zeit, um das Programm zu initialisieren und zu überprüfen, ob eine FreeWare- oder eine lizenzierte Version vorliegt. Darüberhinaus wird bei jedem Start überprüft, ob die Programmdatei im Originalzustand vorliegt: Sollte EUKLID DynaGeo feststellen, dass die EXE-Datei verändert wurde, wird die Programmausführung mit einem entsprechenden Hinweis abgebrochen. Sollte Ihnen das passieren, nehmen Sie die Fehlermeldung bitte sehr ernst: in aller Regel ist dies ein Hinweis darauf, dass Ihr Rechner tatsächlich von einem Virus befallen wurde!


EUKLID DynaGeo liest die Initialisierungsdaten aus zwei Dateien:

  1. Die globalen Standard-Vorgaben werden aus einer Datei namens DYNAGEO.INI gelesen, die im EUKLID DynaGeo-Verzeichnis stehen muss. Gibt es diese Datei dort noch nicht, wird versucht, sie zu erzeugen. Gelingt dies nicht, dann bricht das Programm mit einer entsprechenden Fehlermeldung ab: die DYNAGEO.INI ist für den Programmstart unbedingt notwendig!

  2. Die Benutzer-spezifischen ("privaten") Einstellungen werden aus einer zweiten Initialisierungsdatei namens EDG_USER.INI gelesen. In älteren Windows-Versionen (95, 98, ME) steht diese Datei ebenfalls im DynaGeo-Verzeichnis; auf neueren NT-Versionen wird sie ins Profil des aktuellen Benutzers geschrieben, und zwar ins Verzeichnis %appdata%\dynageo\.

Sofern Sie nichts anderes vorschreiben, wird das Programm versuchen, denselben Zustand wiederherzustellen, in dem der letzte Programmlauf beendet wurde. Dies betrifft die Einstellungen der Bildschirmdarstellung für die geometrischen Objekte (Darstellungsmodus, Punktgröße, Fangbereich usw.) als auch die Menükonfiguration, eine Liste der zuletzt geöffneten GEO-Dateien und die zuvor geladenen Makros.


Wenn diese Prozedur erfolgreich abgeschlossen werden kann, erscheint auf dem Schalter die Aufschrift Okay; ein Klick auf diesen Schalter schließt dann das Titelfenster. Direkt nach dem Programmstart wird das Titelfenster automatisch geschlossen, sobald die Initialisierungen erledigt sind. Nun erst erscheint das eigentliche EUKLID DynaGeo-Fenster.
