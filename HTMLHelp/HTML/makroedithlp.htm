Makrobeschreibung editieren

Mit diesem Befehl können Sie den Hilfetext und den Namen eines Makros editieren. Es können nur Beschreibungen von solchen Makros bearbeitet werden, die gerade geladen sind. Falls mehr als ein Makro geladen ist, wird Ihnen eine Auswahlbox gezeigt, in der sie das gewünschte Makro anklicken können.
