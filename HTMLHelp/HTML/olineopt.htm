Einstellungen | Ortslinien

Minimaler Abstand von Ortslinienstützpunkten:

    Beim Erzeugen einer Ortslinie wird erst dann ein neuer Punkt zur Ortslinie hinzugefügt, wenn sich der erzeugende Punkt hinreichend weit von der letzten registrierten Lage entfernt hat. Hier legen Sie den entsprechenden Mindestabstand (in Pixeln) fest.
    Bereich: [ 1 ........ (10) ........50 ]

Neue Ortslinien als dynamische Objekte erzeugen:

    Ist diese Option angewählt, wird (nach Möglichkeit) eine dynamische Ortslinie erzeugt, die am Zugmodus teilnimmt. Andernfalls wird lediglich eine statische Spur erzeugt.
    Normalerweise: eingeschaltet.

Neue Ortslinien als Kurven darstellen:

    Ist diese Option gewählt, dann wird eine neu erzeugte Ortslinie durch eine Bezier-Kurve durch die aufgezeichneten bzw. errechneten Stützpunkte dargestellt. Andernfalls werden die Stützpunkte selbst dargestellt.
    Normalerweise: eingeschaltet.

Neue Ortslinien falls möglich in Standardkurven konvertieren:

    Ist diese Option gewählt, dann wird bei jeder neu erzeugten Ortslinie geprüft, ob die Stützpunkte auf einer Geraden oder einem Kreis liegen. Wenn dies der Fall ist, fragt EUKLID DynaGeo nach, ob statt der Approximationskurve durch die Stützpunkte im folgenden diese Gerade bzw. der Kreis angezeigt werden soll. Falls Sie diese Frage bejahen, wird bei folgenden Zugoperationen stets diese Gerade bzw. dieser Kreis angezeigt.
    Normalerweise: ausgeschaltet.

Neue Einhüllende als dynamische Objekte erzeugen:

    Ist diese Option angewählt, wird (nach Möglichkeit) eine dynamische Einhüllende erzeugt, die am Zugmodus teilnimmt. Andernfalls wird lediglich eine statische Momentaufnahme der Einhüllenden erzeugt.
    Normalerweise: eingeschaltet.

Neue Einhüllende als Kurven darstellen:

    Ist diese Option gewählt, dann wird eine neu erzeugte Einhüllende durch eine Bezier-Kurve durch die aufgezeichneten bzw. errechneten Stützpunkte dargestellt. Andernfalls werden die Stützpunkte selbst dargestellt.
    Normalerweise: eingeschaltet.

Die Geradenschar einer Einhüllenden nach dem Erzeugen weiterhin anzeigen:

    Ist diese Option gewählt, dann wird bei jeder neu erzeugten Einhüllenden die definierende Geradenschar angezeigt; andernfalls wird nur die Einhüllende selbst gezeichnet. Dies ist im Normalfall zu bevorzugen, weil die große Anzahl zusätzlicher Geraden meist mehr Verwirrung bringt als Nutzen.
    Normalerweise: ausgeschaltet.
