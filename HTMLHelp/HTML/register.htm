Nachträgliche Lizenzierung einer Freeware-Installation


Ob Sie eine lizenzierte Vollversion von DynaGeo haben, die sich mit Ihren persönlichen Daten meldet, oder ob Sie "nur" eine Freeware-Version installiert haben, -- all das hat keinerlei Folgen für den Funktionsumfang des installierten Dynageo-Programms. Trotzdem könnten Sie den Wunsch haben, dass sich "Ihr DynaGeo" bei Ihnen auch als lizenzierte Vollversion zur Arbeit meldet statt als anonyme Freeware. Wie Sie das erreichen können, erfahren Sie hier:

Sie können Ihre persönlichen Lizenzdaten bei einer installierten Freeware-Version jederzeit nachträglich eintragen, wenn Sie nur eine gültige DGL-Datei haben. Die einfachste Methode: Kopieren Sie Ihre DGL-Datei ins EUKLID DynaGeo-Verzeichnis, also z.B. "C:\Programme (x86)\DynaGeo\"). Dazu müssen Sie allerdings mit Administratorrechten angemeldet sein; als gewöhnlicher Benutzer haben Sie nämlich keine Schreibrechte in den Verzeichnissen, in denen Windows die Programme installiert! Wenn Sie Ihre DGL-Datei erfolgreich ins DynaGeo-Verzeichnis kopiert haben, sollte sich das Programm beim nächsten Start als lizenzierte Vollversion melden.

    Sie müssen diese Kopieraktion nicht unbedingt selbst durchführen. Soll EUKLID DynaGeo diese Arbeit für Sie erledigen, so wählen Sie im Hilfe-Menü der Freeware-Version den Eintrag Registrierung; suchen Sie dann in dem Dialog-Fenster nach Ihrer DGL-Datei (mit dem Knopf Durchsuchen). Wenn das Editierfeld den vollständigen Pfad zu einer existierenden DGL-Datei enthält, wird nach einem Klick auf den Knopf Registrieren eine Kopie dieser Datei ins EUKLID DynaGeo-Verzeichnis geschrieben. Wenn sich EUKLID DynaGeo beim nächsten Start als lizenzierte Vollversion meldet, wird der Menüpunkt Registrierung aus dem Hilfe-Menü entfernt: er ist ja nun auch nicht mehr nötig!

Da die DGL-Dateien inzwischen aber faktisch funktionslos sind, was den Betrieb des Programms zur Laufzeit angeht, bitte ich alle, die keine DGL-Datei mehr zur Verfügung haben, sich mit der Freeware-Variante zufrieden zu geben. Ich werde keine DGL-Dateien mehr versenden!

Für den Umgang mit noch existierenden Lizenzdaten gilt weiterhin:

Jede Weitergabe der installierten Vollversion bzw. der Lizenzdaten (also der DGL-Datei!) außerhalb des Geltungsbereiches der jeweiligen Lizenz ist nach wie vor untersagt.

Von der Archiv-Datei hingegen dürfen Sie beliebig viele Kopien herstellen und weitergeben, da sie lediglich eine Freeware-Version (ohne persönliche Lizenzdaten) enthält. Einfacher ist es jedoch, wenn Sie nur die Homepage-Adresse von EUKLID DynaGeo weitergeben:

    http://www.dynageo.de

Dort kann sich jeder Interessent selbst eine Freeware-Version holen, von der er dann auch sicher sein kann, dass sie aktuell ist.
