Punkt auf einer Linie    

(automatisch wiederholbar)

Klicken Sie mit der Maus diejenige Linie an, auf der Sie einen Punkt erzeugen wollen. Mögliche Trägerlinien für den Punkt sind:

  • Gerade Linien (also: Geraden, Halbgeraden und Strecken)
  • Kreise (auch: Kreisbögen)
  • Polygone
  • Ortslinien
Der Punkt wird erzeugt und an diese Linie gebunden.

Sie können den so erzeugten Punkt nachträglich wieder von der Trägerlinie lösen, indem Sie den Menüpunkt Bearbeiten | Bindung eines Punktes lösen aufrufen oder diesen Befehl über das Kontext-Menü des Punktes auswählen.
