Copyright-Vermerk editieren


Dieser Menüpunkt führt zu einem Fenster, in dem Sie einen Copyright-Text für Ihre aktuelle Zeichnung eingeben bzw. bearbeiten können.


  1. In einem ersten Feld "Copyright-Vermerk" können Sie als Besitzer einer EUKLID DynaGeo - Lizenz die von Ihnen erstellten Dateien mit einem Copyright-Vermerk versehen, den nur Sie wieder editieren können. Wenn Sie jedoch eine GEO-Datei laden, die unter einer anderen EUKLID DynaGeo - Lizenz erstellt wurde, dann ist dieses Copyright-Feld schreibgeschützt.

    Achtung !
      Ihr Copyright-Vermerk ist kein Schreibschutz für den Datei-Inhalt! Jeder Nutzer kann sein Exemplar Ihrer Zeichnung beliebig weiterbearbeiten, Objekte editieren oder löschen, andere hinzufügen... - lediglich Ihren Copyright-Vermerk kann er nicht verändern oder gar entfernen: auch die veränderte Datei wird diesen Ihren Vermerk enthalten, selbst wenn sie unter einem anderen Namen abgespeichert wird.


  2. In einem zweiten Feld "Aktuelle Datei-Attribute" werden Ihnen der Dateiname, das Erstellungsdatum der Datei, das Datum der letzten Änderung sowie die EUKLID DynaGeo - Version angezeigt, mit der diese Datei zuletzt bearbeitet wurde. Darüberhinaus wird die aktuelle Dateigröße angezeigt. Diese Daten können hier nicht verändert werden und dienen nur zu Ihrer Information.




Wenn Sie das Fenster mit einem Klick auf den "Okay"-Knopf rechts oben schließen, werden Ihre Änderungen in die aktuelle Zeichnung geschrieben. Schließen Sie das Fenster dagegen mit dem "Abbrechen"-Knopf, werden Ihre Änderungen verworfen.
