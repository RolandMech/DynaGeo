Objekt drehen    

(automatisch wiederholbar)

Geben Sie das abzubildende Objekt, das Drehzentrum und den Drehwinkel an.

Bevor Sie ein Objekt drehen können, muss der zu verwendende Drehwinkel schon in Ihrer Zeichnung vorhanden sein. Es gibt verschiedene Möglichkeiten, wie Sie den Winkel für eine Drehung angeben können:

  • Sie können einen schon in Ihrer Zeichnung vorhandenen Winkel verwenden. Wenn Sie z.B. in einem Dreieck zuvor schon einen Winkel markiert haben, können Sie nun bei der Eingabe des Drehwinkels auf diesen Winkelbogen klicken. Dann wird für ihre Drehung die aktuelle Weite dieses Winkels verwendet.

  • Sie können auf ein schon in Ihrer Zeichnung vorhandenes Zahlobjekt klicken. Dann wird dessen aktueller Wert als Weite des Drehwinkels (im Bogenmaß) verwendet. Wenn Sie den Wert im Gradmaß interpretiert haben wollen, fügen Sie bitte ein Gradzeichen (°) an. Dadurch wird die interne Konversion ins Bogenmaß angestoßen.

  • Sie können auf ein schon vorhandenes Termobjekt klicken. Auch in diesem Fall wird der aktuellen Wert des Termobjekts als Weite des Drehwinkels (im Bogenmaß) verwendet. Der im Termobjekts gespeicherte Term könnte z.B. eine Konstante sein, die Sie zuvor auf den gewünschten Wert einstellen; Sie können aber auch einen komplizierten Term angeben, der von mehreren Objekten Ihrer Zeichnung abhängt. Soll der Termwert als Winkelweite im Gradmaß interpretiert werden, dann fügen Sie zusätzlich ein Gradzeichen hinzu. (Ist der Term eine Summe mit mehreren Summanden, müssen Sie gegebenenfalls den ganzen Term klammern, ehe Sie das Gradzeichen anfügen.)

Nach der Eingabe des abzubildenden Punktes, des Drehzemtrums und des Drehwinkels wird das entsprechende Bildobjekt gezeichnet. Alle Punkt-Bildpunkt-Paare werden durch Spuren verbunden, sofern Sie die automatische Spurerzeugung nicht deaktiviert haben (siehe Einstellungen | Abbildungen).


Bitte beachten Sie:

  • EUKLID DynaGeo interpretiert die Drehwinkel stets mathematisch positiv orientiert; es wird also gegen den Uhrzeigersinn dreht. Wenn Sie eine Drehung im Uhrzeigersinn ausführen wollen, müssen Sie einen negativen Wert für den Drehwinkel angeben.

  • Wenn Sie diese Abbildung automatisch wiederholen, wird für alle Wiederholungen das bei der ersten Drehung benutzte Drehzentrum und der dort benutzte Drehwinkel verwendet. Es werden also alle Objekte um denselben Punkt und um denselben Winkel gedreht!
