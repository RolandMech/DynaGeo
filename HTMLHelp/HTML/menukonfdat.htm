Einstellungen | Menü Konfigurations-Datei

Das lokale Menü im Einstellungen - Fenster werden Sie voraussichtlich eher selten brauchen. Es enthält die folgenden sechs Einträge:

  • Standardeinstellungen laden
    Dieser Befehl setzt alle Einstellungen der Objektdarstellung auf die Standardwerte zurück.

  • Benutzereinstellungen laden
    Hier können Sie die zuvor abgespeicherten benutzerdefinierten Einstellungen des Programms wieder laden. Dies geschieht bei jedem Programmstart zunächst automatisch. Wenn Sie aber (etwa zur Probe) die Einstellungen verändert haben, ohne sie abzuspeichern, können Sie die Veränderungen mit diesem Befehl wieder rückgängig machen.

  • Benutzereinstellungen speichern
    Mit diesem Befehl können Sie Ihre eigenen Einstellungen abspeichern. Beim nächsten Programmstart werden diese dann automatisch gelesen, so dass Sie mit denselben Darstellungs-Optionen arbeiten können wie zuvor.


Die letzten drei Menüpunkte sind nur für Administratoren zugänglich, also für solche Benutzer, die schreibenden Zugriff auf die zentrale Konfigurationsdatei DYNAGEO.INI im DynaGeo-Verzeichnis haben.

  • Aktuelle User-Einstellungen als Standard speichern
    Dieser Befehl erlaubt dem Administrator, seine aktuellen privaten Einstellungen als Standard-Einstellungen für alle Benutzer abzuspeichern. Durch geeignete Netzwerk-Einstellungen kann man dann erzwingen, dass alle Benutzer mit diesen Einstellungen arbeiten müssen.

  • Lizenzdaten aktualisieren
    Mit diesem Befehl können Sie bei einer lizenzierten Vollversion neue Lizenzdaten aus einer DGL-Datei einlesen. Dies ist nur dann nötig, wenn sich Ihre Lizenzdaten geändert haben, z.B. bei einem Upgrade von einer Schullizenz auf eine Erweiterte Schullizenz.

  • Netzwerk-Einstellungen...
    Dieser Befehl führt zu einem Dialog, in dem der Administrator die Vorgaben für alle Benutzer der vorliegenden Version von EUKLID DynaGeo im lokalen Netz festlegen kann. Details zu diesem nicht ganz einfachen Thema finden Sie im Abschnitt Netzwerk-Einstellungen.


Normalerweise werden die gewählten Einstellungen beim Beenden des Programms automatisch gespeichert, so dass Sie die ersten 3 Menüpunkte nur in Ausnahmefällen brauchen werden. Die letzten beiden Menüpunkt sollten Sie ebenfalls sparsam einsetzen: benutzen Sie sie nur, wenn Sie genau wissen, was Sie tun!
