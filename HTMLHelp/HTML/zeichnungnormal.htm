Zeichung auf Originalgröße zurücksetzen

Dieser Menübefehl setzt die Zeichnung mit einer zentrischen Streckung auf die ursprüngliche Größe zurück. Der Streckfaktor wird dabei so gewählt, dass die Länge der Einheitsstrecke wieder genau so lang ist, wie sie auf dem aktuellen Computer für eine neue Zeichnung eingestellt wird. Das Zentrum der Streckung wird so gewählt, dass der Ursprung des Koordinatensystems wieder in der Fenstermitte liegt.

Es werden sowohl das Koordinatensystem als auch alle konstruierten Objekte gezoomt. Lediglich Term- und Zahlobjekte sowie Textboxen, die nicht an ein Objekt gebunden sind, bleiben an ihrem Platz im aktuellen Fenster: diese Objekte werden stets relativ zur linken oberen Ecke des aktuellen Programmfensters positioniert.
