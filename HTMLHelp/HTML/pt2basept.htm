Punkt in freien Basispunkt verwandeln

Mit diesem Befehl können Sie einen zuvor konstruierten Punkt in einen freien Basispunkt verwandeln, wobei alle von diesem Punkt abhängigen Objekte erhalten bleiben. Damit lässt sich eine Zeichnung "verallgemeinern": ein zuvor durch eine Konstruktion festgelegter Punkt kann nun frei bewegt werden.

Klicken Sie mit der linken Maustaste auf den konstruierten Punkt. Dann wird er in einen Basispunkt verwandelt, den Sie mit der Maus frei verziehen können. Wenn Sie statt eines konstruierten Punktes einen an eine Linie gebundenen Basispunkt auswählen, wird die Bindung an die Linie gelöst.


Achtung !

    Bitte beachten Sie, dass es im Allgemeinen keine einfache Möglichkeit gibt, diesen Befehl wieder rückgängig zu machen. Wenn Sie sich Ihrer Sache nicht ganz sicher sind, dann sollten Sie Ihre Zeichnung vor der Anwendung dieses Befehls abspeichern.
