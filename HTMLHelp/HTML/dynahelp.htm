Dynamisches Hilfefenster

Dynamisches Hilfefenster erstellen

Dieser Befehl erstellt ein dynamisches Hilfefenster. Das ist eine Textbox mit erweiterten Fähigkeiten:

  1. Zur Laufzeit erscheint zunächst nur der erste Absatz des gesamten Textes. Durch Klick auf einen kleinen "Pfeil nach unten" unter der letzten Zeile des angezeigten Textes kann der jeweils nächste Absatz sichtbar gemacht werden. So kann der Text also vom Benutzer nach Bedarf "aufgeblättert" werden. Ist das Textende erreicht, wird unter der letzten Textzeile stattdessen ein "Pfeil nach oben" angezeigt; ein Klick darauf verbirgt alle Absätze bis auf den ersten, setzt also die Anzeige in den ursprünglichen Zustand zurück.

    Das jeweilige Absatz-Ende wird zur Design-Zeit definiert, indem mit Hilfe des "Absatz-Ende"-Knopfes eine horizontale Linie in den Text eingefügt wird.

  2. Im Text eines dynamischen Hilfefensters können Links zu anderen GEO-Dateien (bzw. HTML-Dateien) definiert werden. Ein Klick auf den "Link einfügen"-Knopf liefert ein Fenster, in dem Sie den Text des Links sowie die Ziel-Datei angeben können. Der Pfad zur Ziel-Datei muss als relativer Pfad bezüglich der aktuellen GEO-Datei formuliert werden. Befinden sich beide Dateien im selben Verzeichnis, genügt also der Name der Zieldatei.
    Zur Laufzeit führt ein Doppelklick des Benutzers auf den Link dazu, dass die darin angegebene Datei geladen wird.


Ansonsten verhalten sich dynamische Hilfefenster genau wie gewöhnliche Textboxen. Lediglich der automatische Aufruf des Textbox-Editors beim Doppelklick auf ein dynamisches Hilfefenster ist deaktiviert, weil der Doppelklick nun für die Aktivierung eventuell im Hilfetext vorhandener Links gebraucht wird. Um ein vorhandenes dynamisches Hilfefenster zu editieren, können Sie den Befehl "Editieren..." aus dem Kontext-Menü dieses Objekts benutzen.

Ein dynamisches Hilfefenster kann nicht nur zu seinem ursprünglichen Zweck verwendet werden kann: Sie können stattdessen auch die bei der Initialisierung hineingeschriebenen Inhalte komplett löschen und das Fenster dann als ein gewöhnliches Textfenster benutzen. In der Tat ist dies derzeit der einzige Weg, einen Link in einem "normalen" Textfeld unterzubringen!

Wenn Sie einen bestehenden Link editieren wollen, stellen Sie den Cursor auf den Linktext, bevor Sie auf den Knopf Link einfügen klicken. Dann wird das Fenster zur Eingabe des Links mit den Daten dieses schon existierenden Links initialisiert.



Achtung !
  • Wenn Sie in einem dynamischen Hilfefenster einen "aufblätterbaren" Hilfe-Text zur Verfügung stellen wollen, dann muss jedes "logische Blatt" dieses Textes von Absatzende-Markierungen umschlossen sein. DynaGeo kann zwar eine am Ende des gesamten Textes fehlende Absatzende-Markierung selbständig ergänzen; die im laufenden Text benötigten jedoch müssen Sie setzen. Wenn Ihr Text nur eine Absatzende-Markierung ganz an seinem Ende enthält, wird diese entfernt, weil es dann keinen von solchen Markierungen umschlossenen Absatz gibt.

  • Wenn der Text Ihres dynamischen Hilfefensters beim Schließen des Editorfensters weder einen Link noch eine (gültige) Absatzende-Markierung enthält, dann verliert das entsprechende Textobjekt den Status "dynamisch" und wird zu einer "normalen" Textbox. Wenn Sie diese Textbox später nochmals editieren (mit dem Befehl "Editieren..." aus dem Kontext-Menü), dann werden die Befehlsknöpfe Link einfügen und Absatz-Ende nicht mehr zur Verfügung stehen.
