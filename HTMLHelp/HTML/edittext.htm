Textbox editieren

Mit diesem Befehl können Sie den Text einer Textbox nachträglich bearbeiten: wenn Sie auf den zu editierenden Text klicken, dann erscheint das Editier-Fenster, und der Inhalt der Textbox wird zur Bearbeitung geladen.

Sie brauchen allerdings nicht immer den umständlichen Weg über das Menü zu gehen, wenn Sie eine Textbox editieren wollen: ein Doppelklick mit der linken Maustaste auf den Text bringt ebenfalls das Editierfenster zur Anzeige!
