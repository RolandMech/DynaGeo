Menü Makro

Dieses Menü dient der Makro-Verwaltung. Makros können erstellt, exportiert, wieder importiert und (natürlich) ausgeführt werden. Wenn Sie ein neues Makro erstellen oder eines aus einer Makrodatei importieren, dann wird das Makro-Menü um jeweils eine Zeile verlängert, die den Namen des hinzugefügten Makros zeigt. Wenn Sie diese Menüzeile anklicken, wird das Makro ausgeführt.
