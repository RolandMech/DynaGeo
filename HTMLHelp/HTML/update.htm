Wie bekomme ich ein Update ?

Wenn Sie ein Update wollen, genügt es, dass Sie eine neue Version über Ihre alte Version ”drüberinstallieren”. Sollten Sie vorher eine lizenzierte Vollversion gehabt haben, dann bleiben Ihre Lizenzdaten dabei erhalten. Eine "neue Version" können Sie am einfachsten von meiner Internet-Homepage holen. Die Adresse ist:

http://www.dynageo.de

Außerdem finden Sie dort eine Menge Wissenswertes und manch Aktuelles rund um EUKLID DynaGeo, so dass Sie ohnehin ab und an auf diese Seiten surfen sollten. Inzwischen gibt es auch die Möglichkeit, sich dort in eine Mailing-Liste einzutragen; dann werden Sie über wichtige Neuerungen automatisch unterrichtet.
