Funktions-Schaubild    

Wenn Sie diesen Befehl anwählen, öffnet sich ein kleines Fenster, in dem Sie den Term der gewünschten Funktion eingeben können. Nach einem abschließenden Klick auf Okay verschwindet das Fenster. EUKLID DynaGeo erstellt ein entsprechendes Schaubild-Objekt und zeichnet das Schaubild Ihrer Funktion ins Koordinatensystem der Zeichnung ein.


Bemerkungen:

  1. Innerhalb des Funktionsterms können Sie alle üblichen mathematischen Standard-Funktionen benutzen. Darüberhinaus kann der Funktionsterm auch Referenzen auf Objekte Ihrer Zeichnung enthalten. Genaueres zu diesem Thema finden Sie auf der Seite Die Syntax von Termen und den darauf folgenden Seiten.

  2. Obwohl EUKLID DynaGeo alle für die Zeichnung relevanten Winkelweiten stets im Bogenmaß berechnet, können Sie in einem Funktionsterm eine Winkelweite auch im Gradmaß eingeben, indem Sie nach der Angabe der Winkelweite ein Gradzeichen (°) hinzufügen. Dadurch wird die interne Konversion der eingegebenen Winkelweite ins Bogenmaß angestoßen. Weitere Details zu diesem Thema finden Sie hier: Das Problem der Winkelmaß-Einheit.

  3. Der Zeichenbereich für ein Funktions-Schaubild ist stets das gesamte sichtbare Intervall der x-Achse. Wenn der Definitionsbereich der Funktion nur Teile davon abdeckt, dann wird das Schaubild nur in denjenigen Bereichen dargestellt, in denen EUKLID DynaGeo gültige Werte des Funktionsterms findet.

  4. Das Kontextmenü eines Funktions-Schaubilds zeigt neben den üblichen Einträgen zusätzlich die folgenden Menüpunkte:

    • Farbe...:
        hier können Sie eine beliebige Zeichenfarbe wählen;

    • Linienart...:
        hier können Sie einen Linienstil wählen;
        dieser Punkt kann bei manchen Funktionen fehlen (weitere Details dazu).

    • Funktionsterm editieren:
        hier können Sie den Funktionsterm nachträglich bearbeiten.

    • Wertetabelle anzeigen:
        ruft ein Fenster auf, das eine Wertetabelle dieser Funktion anzeigt. Voreingestellt ist ein x-Bereich, der dem aktuell im Zeichenfenster sichtbaren x-Bereich entspricht, und die Schrittweite 1. Sie können jedoch sowohl die Grenzen des x-Bereichs als auch die Schrittweite in diesem Fenster frei skalieren. Zusätzlich zu den Werten der ausgewählten Funktion können Sie in weiteren Spalten die Werte aller anderen Funktionen der aktuellen Zeichnung anzeigen. Die Fenstergröße ist veränderbar; wenn die Tabelle nicht ins Fenster passt, wird ein entsprechender Rollbalken angezeigt.
        Die Tabelle hat ebenfalls ein Kontextmenü, über das Sie z.B. den Tabelleninhalt in die Zwischenablage kopieren können. Die anderen Menü-Einträge sind selbsterklärend: "Tabelle aktualisieren" und "Fenster schließen".

ALT-Variante: Interpolations-Polynom-Funktion

    Wenn Sie den Befehl Funktions-Schaubild über die Werkzeugleiste anwählen und beim Klicken auf das entsprechende Icon die ALT-Taste gedrückt halten, dann erwartet der Befehl statt eines Funktionsterms eine Liste von schon existierenden Punkten, durch die das gewünschte Schaubild hindurchgehen soll. Intern wird dazu ein Interpolations-Polynom f errechnet, das genau durch die Punkte Pi(xi,yi) verläuft, also f(xi) = yi liefert für alle Punkte der Liste. Details über die Eingabe der Kontrollpunkte finden Sie unter Interpolations-Polynom-Funktion.
