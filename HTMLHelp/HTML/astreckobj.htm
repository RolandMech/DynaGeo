Objekt zentrisch strecken    

(automatisch wiederholbar)

Geben Sie das abzubildende Objekt, das Streckzentrum und den Streckfaktor an.

Bevor Sie ein Objekt strecken können, muss der Streckfaktor in Ihrer Zeichnung vorhanden sein. Es gibt verschiedene Möglichkeiten, wie Sie den Streckfaktor angeben können:

  • Sie können ein schon in Ihrer Zeichnung vorhandenes Zahlobjekt anklicken. Dann verwendet EUKLID DynaGeo dessen aktuellen Wert als Streckfaktor für Ihre zentrische Streckung.

  • Sie können ein schon in Ihrer Zeichnung vorhandenes Termobjekt anklicken. Auch hier wird der aktuelle Wert als Streckfaktor verwendet. Im einfachsten Fall kann das Termobjekt eine Konstante enthalten, die Sie zuvor auf den gewünschten Streckfaktor eingestellt haben. Sie können aber auch einen komplizierten Term verwenden, der von mehreren Objekten Ihrer Zeichnung abhängt.

Es wird das entsprechende Bildobjekt gezeichnet. Alle Punkt-Bildpunkt-Paare werden durch Spuren verbunden, sofern Sie die automatische Spurerzeugung nicht deaktiviert haben (siehe Einstellungen | Abbildungen).


Bitte beachten Sie:

    Wenn Sie diese Abbildung automatisch wiederholen, wird für alle Wiederholungen das bei der ersten Zentrischen Streckung benutzte Zentrum und der dort benutzte Streckfaktor verwendet. Es werden also alle Objekte an demselben Zentrum mit demselben Faktor gestreckt!
