Riemann-Summen

Dieser Befehl zeichnet eine "Riemann-Summe", also eine Serie von (gleichbreiten) Rechtecken, die die Integralfläche zwischen einem Funktionsschaubild und der x-Achse approximiert. Um die gewünschte Fläche festzulegen, führen Sie bitte die folgenden drei Schritte aus:

  1. Geben Sie zunächst das Funktions-Schaubild an, das die gewünschte Fläche nach oben begrenzt.

  2. Markieren Sie danach das Integrations-Intervall: geben Sie dazu zwei Punkte an, deren x-Koordinaten das Integrations-Intervall bestimmen. Sie können entweder schon vorhandene Punkte benutzen oder neue Basispunkte durch einen Mausklick erzeugen. Sind die Begrenzungspunkte des Integrations-Intervalls ungebundene Basispunkte, dann werden sie bei der Erzeugung der Fläche an die x-Achse gebunden.

  3. Schließlich erscheint ein Fenster, in dem Sie den Typ der Riemann-Summe (Ober- oder Untersumme) und die Anzahl der Teilintervalle eingeben können. Diese Anzahl kann auch durch einen Term festgelegt werden, der eine Referenz auf ein Zahlobjekt enthält; dann können Sie die Anzahl der Teilintervalle später bequem dynamisch ändern. Die minimale Anzahl der Teilintervalle ist 1, die maximale 256.

Danach wird ein entsprechendes Riemann-Summen-Objekt erzeugt und auf dem Bildschirm angezeigt.


Im Kontext-Menü einer Riemann-Summe finden Sie neben den für Flächen-Objekte typischen Einträgen zusätzlich noch den folgenden Menüpunkt:
  • Anzahl der Teilintervalle ändern...
    Wenn Sie diesen Menüpunkt anklicken, erscheint wieder das Fenster zur Eingabe der Anzahl der Teilintervalle der Riemann-Summe. So können Sie diese Anzahl nachträglich ändern, selbst wenn Sie bei der Erzeugung des Riemannsummen-Objekts als Anzahl eine Konstante eingegeben haben.



Beachten Sie bitte:
  • Wenn Ihre Randfunktion f nicht überall positive Werte hat, mag es sein, dass die von DynaGeo gezeichnete Untersumme nicht ganz Ihren Erwartungen entspricht. Zur Klarstellung sei hier nochmals explizit darauf hingewiesen, dass für die Untersummen in jedem Teilintervall als (vorzeichenbehaftete!) Höhe des Rechteck-Streifens jeweils der kleinste Wert genommen wird, den die Funktion f in diesem Teilintervall annimmt, bei den Obersummen entsprechend der größte Wert. Nur so ist gewährleistet, dass stets gilt:

    U(f,a,b) <= I(f,a,b) <= O(f,a,b)

    wobei U der Wert der Untersumme ist, I der Wert des Integrals und O der Wert der Obersumme.

  • Um den Aufwand bei der numerische Berechnung einer Riemannsumme auf ein vernünftiges Maß zu begrenzen, ist die maximale Anzahl der Teilintervalle auf 256 festgesetzt. Wenn Sie eine größere Anzahl anfordern, wird DynaGeo trotzdem intern stets mit höchstens 256 Teilintervallen rechnen. Sollte Ihnen diese Grenze zu eng sein,können Sie die maximal mögliche Anzahl der Teilintervalle unter "Verschiedenes | Einstellungen | Messen" höher setzen.
