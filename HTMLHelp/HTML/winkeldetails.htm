Weitere Details zu Winkeln

Sie können den Radius des Winkelbogens verändern, indem Sie im Zugmodus daran ziehen.

Ist der Winkel ein Rechter, wird er besonders markiert: statt des üblichen Winkelbogens wird ein rechtwinkliger Haken gezeigt. Falls Ihnen dies nicht zusagt und Sie die traditionelle Rechtwinkelmarkierung mit dem Punkt bevorzugen, müssen Sie in den Einstellungen auf der Registerkarte Darstellung die Option "Rechten Winkel mit Punkt markieren" anwählen. Außerdem können Sie hier einstellen, ob Sie "Winkelmarkierungen gefüllt darstellen" wollen oder nicht. Normalerweise ist diese Option markiert.

Auf der Registerkarte Internes lässt sich darüberhinaus noch einstellen, in welchem Intervall um den exakten rechten Winkel ein gegebener Winkel als Rechter angesehen wird.

Auf der Registerkarte Messen können Sie wählen, ob EUKLID DynaGeo die Orientierung eines Winkels berücksichtigen soll. Falls ja, wird von allen Winkeln, die Sie eingeben, angenommen, dass sie mathematisch positiv, also im Gegenuhrzeigersinn orientiert sind. So lassen sich auch überstumpfe Winkel darstellen. Falls Sie die Orientierung der Winkel ausschalten, wird bei jedem eingegebenen Winkel die Reihenfolge der Schenkel automatisch so gewählt, dass der Winkel nicht überstumpf ist.

Wenn Sie Winkel gefüllt darstellen lassen, ist die Füllung eines Winkelbogens für die Maus unsichtbar. Damit haben Sie auf diese Füllung keinen direkten Zugriff mehr. Sie können jedoch die Füllfarbe eines Winkels verändern, indem Sie die Farbe des Winkelbogens verändern:

  • wenn der Winkelbogen schwarz ist, dann wird das Segment gelb gefüllt;
  • wenn der Winkelbogen eine andere Farbe als schwarz hat, dann wird das Segment in einer helleren Variante dieser Farbe gefüllt.
Der "schwarz-gelbe Standardfall" stellt die einzige Ausnahme von dieser letzten Regel dar. Dies wurde so implementiert, weil ein "schwarz-grauer Standardfall" dann doch zu gar zu freudlosen Zeichnungen geführt hätte ;-)


Achtung !

    Winkelbögen sind für EUKLID DynaGeo keine Objekte, d.h. sie können nicht als Eltern weiterer Objekte verwendet werden. Genau wie Namen, Maßanzeigen und Textboxen dienen sie lediglich der Dekoration.
