Objekt löschen    

(automatisch wiederholbar)

Sie müssen zunächst das zu löschende Objekt mit der Maus anklicken. Wenn es andere Objekte gibt, die von dem zu löschenden Objekt abhängen, weil sie nämlich mit seiner Hilfe konstruiert wurden, dann werden das Objekt und diese seine Nachkommen grün markiert, und Sie werden gefragt, ob Sie diese Objekt wirklich alle löschen wollen. Falls ja, wird das Löschen durchgeführt, andernfalls nicht.

Wenn Sie ein Objekt löschen wollen, das keine Nachkommen hat, dann wird die Löschung sofort und ohne Nachfrage durchgeführt, nachdem Sie das Objekt angeklickt haben.

Wenn Sie das Koordinatensystem löschen wollen, erscheint eine Meldung, die Sie darüber informiert, dass das Koordinatensystem nicht gelöscht, sondern nur verborgen werden kann; gelöscht werden lediglich die Fixierungen von Koordinatenpunkten. Wenn Sie dies wollen, dann werden alle Koordinatenpunkte in Basispunkte umgewandelt und anschließend das Koordinatensystem verborgen. Wollen Sie die Koordinatenpunkte als solche erhalten, müssen Sie das Koordinatensystem über den Befehl Objekt verbergen / anzeigen verbergen.
