Die Konstruieren-Leiste

Die Konstruktionsleiste enthält folgende Symbole:

    Einen Punkt erzeugen.
    Einen Punkt mit Koordinaten (x|y) erzeugen.
    Einen Punkt auf einer Linie erzeugen.

    Den Schnitt zweier Linien konstruieren.
    Einen Streckenmittelpunkt erzeugen.

    Eine Strecke zwischen 2 Punkten erzeugen.
    Eine Strecke mit fester Länge erzeugen.
    Einen Vektor erzeugen.
    Eine Halbgerade erzeugen.

    Eine Gerade durch 2 Punkte erzeugen.
    Eine Orthogonale erzeugen.
    Eine Parallele erzeugen.
    Eine Mittelsenkrechte erzeugen.
    Eine Winkelhalbierende erzeugen.
    Eine Gerade in bestimmtem Winkel erzeugen.

    Einen Kreis um einen Mittelpunkt durch einen Kreispunkt erzeugen.
    Einen Kreis mit bestimmtem Radius erzeugen.
    Einen Kreisbogen erzeugen.

    Ein Dreieck erzeugen.
    Ein N-Eck erzeugen.
