Funktionen-Lupe    

Dieser Befehl startet eine "Funktionen-Lupe", die zur Untersuchung lokaler Eigenschaften einer gegebenen Funktion eingesetzt werden kann. Dazu müssen die folgenden Voraussetzungen erfüllt sein:

  1. Die Zeichnung muss ein Funktions-Schaubild enthalten.

  2. Es muss einen (Basis-) Punkt geben, der an das Schaubild gebunden ist, so dass er sich beim Verziehen stets nur auf diesem Schaubild bewegt.

Wenn diese Voraussetzungen erfüllt sind, können Sie eine Funktionen-Lupe erzeugen. Sei nun also f die Funktion, deren Schaubild angezeigt wird, und A der Punkt, der an das Schaubild gebunden ist.

Wenn Sie nun den Befehl "Funktionen-Lupe" auswählen und den Punkt A anklicken, dann geschieht folgendes:

  1. Zunächst wird der Punkt A mit einem "Lupen-Quadrat" der Kantenlänge 2*h umfasst. Das Quadrat wird im Folgenden stets so positioniert, dass A immer der Diagonalenschnittpunkt des Quadrats ist. Die Kantenlänge hingegen ist veränderlich: am Anfang ist h = 1 cm, also die (Anfangs-) Kantenlänge 2 cm.

  2. Sodann wird ein zweites Fenster geöffnet, das den eigentlichen "Blick durch die Lupe" liefert: in diesem "Lupen-Fenster" wird der Inhalt des Lupen-Quadrats vergrößert dargestellt.

  3. Außerdem werden zwei Punkte Ali(xA-h | f(xA-h)) und Are(xA+h | f(xA+h)) erzeugt, die beide stets auf dem Schaubild von f liegen.

  4. Schließlich wird in der linken oberen Ecke des DynaGeo-Zeichenbereichs ein Schieberegler eingeblendet, mit dem der Wert von h für das obige Lupen-Quadrat bzw. die Punkte Ali und Are eingestellt werden kann. Dieser Schieberegler dient zur Steuerung des eigentlich "Zoom-Prozesses": h kann etwa im Bereich [0,0001....1,0] Längeneinheiten variiert werden.
Durch Verschieben von A auf dem Schaubild von f können Sie nun das Zoom-Zentrum wählen, und am h-Schieberegler können Sie die gewünschte Vergrößerung einstellen. Weitere Details zum didaktisch sinnvollen Einsatz der Funktionen-Lupe finden Sie im Abschnitt Details zur Funktionen-Lupe.


Im Kontextmenü des Lupen-Quadrats finden Sie die folgenden Befehle:

  • Sekanten-Steigungen zeigen / entfernen:
      blendet die Sekanten und die Werte Ihrer Steigungen ein bzw. aus;

  • Sekantensteigungs-Funktionen zeigen / entfernen:
      blendet die rechtsseitige und die linksseitige Sekantensteigungs-Funktion ein bzw. aus.

  • Krümmungskreis zeigen / entfernen:
      blendet einen Kreis durch die 3 Punkte Ali, A und Are ein bzw. aus;

  • Schmiege-Parabel zeigen / entfernen:
      blendet eine Parabel durch die 3 Punkte Ali, A und Are ein bzw. aus;
