Animations-Parameter einstellen    

Dieser Dialog bietet Ihnen die Möglichkeit, die Parameter einer Animation zu editieren. Dazu stehen Ihnen die folgenden Datenfelder zur Verfügung:


Steuerobjekt:
    Hier wird Ihnen eine Liste der möglichen Steuerobjekte angeboten, aus der Sie das gewünschte Objekt auswählen können. Erst wenn ein Steuerobjekt gewählt wurde, können weitere Parameter für die Animation gesetzt werden.

Untere Grenze und Obere Grenze:
    Die Animation wird durch einen Positionsparameter gesteuert. Diese beiden Felder zeigen die Grenzen des Intervalls, in dem die Werte für den Positionsparameter liegen. Sie sind normalerweise mit Standardwerten fest vorbelegt; nur wenn das Steuerobjekt ein Zahlobjekt ist, können Sie diese Werte verändern.

Schrittweite:
    Hier stellen Sie ein, um welchen Betrag der Positionsparameter bei jedem Animationsschritt verändert wird. Damit können Sie die Anfangsgeschwindigkeit der Animation wählen. Genaueres dazu finden Sie unter Grundsätzliches zu Aminationen.
