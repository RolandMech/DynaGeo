Wie bekomme ich weitere Hilfe bei schwierigeren Problemen?

Falls Ihre Version von EUKLID DynaGeo ein seltsames Betriebsverhalten oder irgendwelche Fehler zeigt, sollten Sie zunächst in der Online-Hilfe nachschauen, ob Sie dort etwas Erhellendes zu Ihrem Problem finden. Speziell möchte ich hier auf die Abschnitte Probleme und Lösungen hinweisen, aber auch auf das Kapitel "Ein paar Blicke hinter die Kulissen", in dem grundsätzliche Konzepte und Strukturen von DynaGeo dargestellt werden. Fortgeschrittene Benutzer finden im Kapitel "Berichte aus der Programmierwerkstatt" eine Reihe kleiner Aufsätze zu prinzipiellen Problemen der "Dynamischen Geometrie".

Weitere Möglichkeiten haben Sie, wenn Sie auf der DynaGeo-Homepage nachschauen, ob dort etwas zu Ihrem Problem geschrieben steht: im "Forum" gibt es einen Bereich mit "Häufig gestellten Fragen" sowie eine Ecke über "Fehler und Probleme".

Falls Sie dort auch nicht fündig werden, können Sie mir eine möglichst genaue Problembeschreibung per (EMail) schicken; gegebenenfalls mit einer als Anhang beigefügten GEO-Datei (oder auch mehreren, dann bitte "gezippt"!), welche das Problem illustriert.

Im Notfall gibt es auch noch die "Gelbe Post": Sie können mir auch einen Brief schreiben:

    Roland Mechling, Fuchshaldeweg 24a, 77654 Offenburg

Statt über die Post können Sie den Brief auch via Fax (0781 - 43268) schicken. Lassen Sie mich auf jeden Fall wissen, auf welchem Weg ich Ihnen eine Antwort zukommen lassen kann; Sie können mir z.B. auch Ihre Telefon- oder eine Fax-Nummer angeben.

In jedem Fall werde ich mich bemühen, Ihnen zu helfen, so schnell es mir möglich ist.
