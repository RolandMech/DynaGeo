Koordinatensystem    

Dieser Befehl ruft ein Dialogfenster auf, das Ihnen verschiedene Darstellungsmöglichkeiten für das Koordinatensystem anbietet:

In einem Teilfenster "Form" können Sie über das äußere Erscheinungsbild der Koordinatensystems entscheiden:
  • Sie können zunächst wählen, ob der Ursprung sichtbar oder verborgen sein soll.

  • Wenn Sie sich für einen sichtbaren Ursprung entscheiden, können Sie nun wählen, ob die Achsen sichtbar oder verborgen sein sollen.

  • Im Falle sichtbarer Koordinatenachsen können Sie wählen,
    • ob die Achsen mit Zahlen beschriftet werden sollen oder nicht, und
    • ob zusätzlich zu Achsen und Ursprung auch noch Gitterpunkte markiert werden sollen oder nicht.

  • Falls Sie Gitterpunkte anzeigen lassen wollen, können Sie die Dichte der Gitterpunkte verringern, indem Sie für den relativen Abstand zweier benachbarter angezeigter Gitterpunkte statt 1 nun 2 oder 5 wählen.
    Beachten Sie aber, dass sich diese Einstellung nur auf die Darstellung des Koordinatensystems auswirkt! Wenn Sie das Koordinatensystem skalieren wollen, müssen Sie den Zoom-Befehl (Menüpunkt Bearbeiten | Zeichnung...|...vergrößern/verkleinern) benutzen.

  • Im Falle eines sichtbaren Koordinatensystems können Sie außerdem die Bezeichnungen der Achsen editieren. Wenn Sie hier nichts eintragen, werden keine Bezeichnungen an den Achsen angezeigt. Für die meisten Fälle sollten die voreingestellten Achsenbezeichnungen "x" und "y" genügen.

In einem weiteren Teilfenster namens "Achsen-Skalierung" können Sie die Skalierung des Koordinatensystems verändern. Normalerweise sollten Sie die Standard-Einstellung (isotrope Darstellung, also gleiche Einheit auf beiden Achsen) nicht verändern müssen. Sollten Sie sich dennoch für eine anisotrope Darstellung entscheiden, werden Sie schwer verständliche Winkel- und Längenmaße zu verkraften haben. Daher ist das wahrscheinlich wichtigste Element in diesem Teilfenster der Knopf "Skalierung auf Standardwerte zurücksetzen".
