Standardfont für Beschriftungen...

    Diese Schaltfläche führt zu einem Font-Auswahl-Dialog, mit dem Sie den Standard-Font für die Objektnamen vorgeben können. Obwohl hier die Manipulation sämtlicher Font-Parameter möglich ist, sollten Sie keine zu exotischen Einstellungen wählen, da diese auf einem anderen Rechner eventuell nicht realisiert werden können. Üblicherweise genügt es, die Fontgröße den eigenen Bedürfnissen anzupassen.

Keine Namen- und Maßobjekte im Konstruktionstext

    Wenn Sie diese Option anwählen, werden im automatisch erzeugten Konstruktionstext nur die ”echten” geometrischen Objekte aufgeführt, nicht jedoch Namen-Objekte sowie Längen- bzw. Winkelmaß-Objekte. Andernfalls erhalten Sie eine vollständige Liste aller geometrischen Objekte Ihrer Konstruktion.

Darstellung der Geradengleichungen:

    EUKLID DynaGeo kann hinter dem Namen einer Geraden auch deren Gleichung anzeigen. In dieser Auswahlbox können Sie entscheiden, in welcher Form diese Geradengleichungen erscheinen sollen:

    • y = mx + c:
      Dies ist die Standardform für Geradengleichungen im Mathematikunterricht der Mittelstufe. Leider lassen sich damit bekannterweise keine Geraden darstellen, die parallel zur y-Achse verlaufen; falls nötig, wird daher auf eine Darstellung in der Form x = a umgeschaltet.

    • ax + by = c:
      Diese Form der Geradengleichung wird vorwiegend in der Analytischen Geometrie der Oberstufe verwendet.
