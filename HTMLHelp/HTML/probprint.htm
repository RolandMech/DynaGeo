Probleme beim Ausdrucken

a) Falsche Linienstile:

Auf manchen Druckern werden gestrichelte oder punktierte Linien durchgezogen wiedergegeben. In solchen Fällen kann EUKLID DynaGeo beim Ausdruck den Linienstil intern nachbilden, indem statt einer Linie viele kleine Linienstücke ausgegeben werden. Wenn Ihr Drucker diese Unterstützung benötigt, müssen Sie in "Verschiedenes | Einstellungen" auf der Registerkarte "Export" die Schalter "Linienstil intern bilden beim Drucken" bzw. "Kurvenstil intern bilden beim Drucken" aktivieren.


b) Falsche Flächenfüllungen:

Mit Flächenfüllungen gehen manche Drucker noch unprofessioneller um als mit Linienstilen. Speziell die gestrichelten Füllungen von Windows werden auf vielen Druckern nicht ordentlich ausgedruckt: häufig wird die Fläche voll gefüllt dargestellt. Falls der Ausdruck von Flächenfüllungen nicht Ihren Erwartungen entspricht, dann sollten Sie ein Update Ihres Druckertreibers in Erwägung ziehen. Selbst bei renommierten Druckerherstellern kommt es gelegentlich vor, dass die Druckertreiber erst auf Kundendruck von solchen Bugs befreit werden.
