DynaGeoJ und Windows

    DynaGeoJ ist ein ganz gewöhnliches JAVA-Applet und wird als solches weitgehend automatisch von Ihrem WINDOWS-System verwaltet. Wenn der Internet Explorer eine HTML-Seite anzeigen soll, für die DynaGeoJ benötigt wird, dann wird dieses Applet gemäß den Angaben im <APPLET>-Tag des HTML-Quelltextes geholt - entweder über das Netz, wenn eine URL angegeben ist, oder aus dem lokalen Dateisystem, wenn das Tag nur einen relativen Pfad enthält. Das Applet wird gestartet, holt sich die darzustellende Zeichnung aus der im <APPLET>-Tag angegebenen GEO-Datei und stellt sie schließlich im vorgesehenen Fenster in der Webseite dar.

    Obwohl bei der Installation von DynaGeo in der Regel eine Version des DynaGeoJ-Applets im Viewer-Verzeichnis hinterlegt wird, kann es trotzdem sein, dass das das JAR-Archiv neu heruntergeladen wird: auf welchem Weg der Internet Explorer zu einer Instanz von DynaGeoJ kommt, wird ausschließlich anhand der Daten im <APPLET>-Tag entschieden.
