Reguläres N-Eck

(automatisch wiederholbar)

Nach dem Auswählen dieses Befehls erscheint zunächst ein Fenster, in dem Sie die gewünschte Eckenzahl angeben können. Nach dem Schließen des Fensters geben Sie zwei benachbarte Eckpunkte des gewünschten regulären Polygons an. Sie können schon vorhandene Punkte benutzen oder durch Klicken auf eine freie Stelle der Zeichnung neue Punkte erzeugen.

Dann wird das reguläre Polygon gezeichnet. Es wird dabei so orientiert, dass es links von der Seite liegt, die vom ersten der beiden angegebenen Eckpunkte zum zweiten führt. Dies ist übrigens auch der Grund, dass Sie hier nicht einfach eine Strecke anklicken können, die als Seite verwendet werden soll: Sie hätten dann keine Kontrolle über die Orientierung dieser Seite, und das regüläre N-Eck würde möglicherweise auf der falschen Seite erzeugt werden!

Wenn Sie in den "Einstellungen" auf der Registerkarte "Darstellung" die Option "Neue Polygone beim Erzeugen füllen" aktiviert haben, dann erscheint das neu erzeugte reguläre N-Eck gefüllt.
