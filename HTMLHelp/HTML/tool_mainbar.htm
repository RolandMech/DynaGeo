Die Hauptleiste

    Neue Zeichnung

    Zeichnung speichern
    Zeichnung drucken

    Einen Konstruktionsschritt zurück
    Letzte Löschung widerrufen
    Objekt benennen
    Objekt verbergen / anzeigen
    Objekt löschen

    Punkt an Linie binden
    Bindung eines Punktes lösen

    Hintergrund-Bild laden
    Neue Textbox erstellen

    Letztes Makro wiederholen

    Korrektheit überprüfen

    GeoDreieck anzeigen / verbergen
