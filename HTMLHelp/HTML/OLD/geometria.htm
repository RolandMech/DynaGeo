Geometria von Dr. Timo Ehmke

Geometria ist ein JAVA-Programm, mit dem Sie interaktive HTML-Seiten zur Geometrie erstellen können. Die Vielfalt der verfügbaren Funktionen geht weit über das hinaus, was der DynaGeoX-Viewer kann. So bietet Geometria z.B. dynamische Veränderungen der Zeichnung in Abhängigkeit von Benutzereingaben sowie die Möglichkeit, Animationen geometrischer Zeichnungen zu programmieren.

Aufgrund der Unterschiede in den Datenstrukturen von Geometria und EUKLID DynaGeo sind derzeit beim Export die folgenden Inkompatibilitäten zu beachten:

  • Strecken fester Länge werden von Geometria nicht unterstützt; sie werden daher als gewöhnliche Strecken zwischen 2 Punkten exportiert.

  • Es sind nur Abstände eines Punktes zu einem anderen Punkt oder einer Geraden oder einem Kreis erfolgreich exportierbar. Zudem muss im letzten Fall der Punkt außerhalb des Kreises liegen.

  • Von den Linienattributen werden nur die Farben korrekt übernommen. Linienstile (wie fett oder gestrichelt) werden von Geometria nicht unterstützt, daher werden alle Linien als dünne, durchgezogene Linien exportiert.

  • Teilverhältnisse werden derzeit von Geometria noch nicht unterstützt. Bitte vermeiden Sie Terme, die Teilverhältnisse enthalten.

Die meisten EUKLID DynaGeo - Zeichnungen werden sich aber wohl ohne größere Probleme im Geometria-Format exportieren lassen. Herr Ehmke stellt das Programm als Freeware zur Verfügung. Weitere Details können auf seinen Seiten im Internet unter

    http://www.geometria.de/

nachlesen. Sie finden dort eine ausführliche Dokumentation zu Geometria, einige Anwendungsbeispiele und einen Link zu den Web-Seiten des Instituts für Mathematik der Universität Flensburg: dort gibt es eine umfangreiche Galerie interessanter geometrischer Fragestellungen, die mit Geometria veranschaulicht werden.

Ehe Sie mit EUKLID DynaGeo HTML-Seiten für Geometria herstellen, sollten Sie sicherstellen, dass eine JAVA-VM auf Ihrem Rechner installiert ist. Außerdem sollten Sie sich die JAVA-Datei "geometria.jar" von der Geometria-Homepage herunterladen. Kopieren Sie diese dann in das Unterverzeichnis "Viewer" Ihres EUKLID DynaGeo - Verzeichnisses.
