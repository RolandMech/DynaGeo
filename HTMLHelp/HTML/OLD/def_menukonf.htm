Menükonfiguration:

Eine Menükonfiguration ist eine bestimmte Auswahl von aktiven Befehlen aus der Gesamtheit der möglichen EUKLID DynaGeo - Menübefehle. Sie können den Befehlsumfang des Programms reduzieren, indem Sie einige Befehle deaktivieren.

Deaktivierte Menübefehle erscheinen grau (statt schwarz) und können nicht mehr angewählt werden; falls zu einem deaktivierten Menübefehl auch ein Icon in einer Werkzeugleiste gehört, dann wird dieses Icon entfernt.
