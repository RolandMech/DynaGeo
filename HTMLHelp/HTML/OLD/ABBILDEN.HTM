Die Abbilden-Leiste

Die Abbilden-Leiste enthält folgende Symbole:

    Eine Spiegelachse festlegen.
    Ein Objekt an einer Achse spiegeln.
    Ein Symmetriezentrum festlegen.
    Ein Objekt an einem Punkt spiegeln.

    Einen Verschiebungsvektor festlegen.
    Ein Objekt verschieben.

    Einen Drehwinkel festlegen.
    Ein Objekt drehen.

    Einen Punkt an einem Kreis spiegeln.
