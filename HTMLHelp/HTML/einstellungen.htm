Einstellungen

Dieser Dialog dient zum Verändern einer Vielzahl von Parametern, mit deren Hilfe Sie das Programm an Ihre speziellen Bedürfnisse anpassen können. Sofern für einen Parameter Werte eingegeben werden können, ist bei der Beschreibung auf den folgenden Seiten stets ein Wertebereich angegeben, und zwar in dem Format:

    [ Minimum.....(Standardwert).....Maximum ]
Die nach einer Neuinstallation eingestellten Standard-Werte sind für die meisten Anwendungsfälle ausreichend. Sie sollten den Wert eines Parameters nur ändern, wenn Sie genau wissen, was Sie tun.

Aufgrund der inzwischen recht großen Zahl von Einstellmöglichkeiten sind diese auf die folgenden Registerkarten verteilt:


    Start-Optionen
    Darstellung
    Messen
    Internes
    Menü-Konfiguration
    Texte
    Export
    Abbildungen
    Ortslinien

Am oberen Rand des Fensters erscheint außerdem ein lokales Menü, das dem Laden und Speichern der Einstellungen dient:


    Menü Konfigurations-Datei
