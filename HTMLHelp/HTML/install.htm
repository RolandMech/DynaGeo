Installation auf Einzelarbeitsplätzen


Laden Sie sich das Installations-Archiv (namens "edg40.exe" oder ähnlich) für EUKLID DynaGeo von der DynaGeo-Homepage herunter und speichern Sie es auf Ihrem Rechner, wo immer Sie wollen. Starten Sie dann die Installation mit einem Doppelklick im Explorer auf diese Archiv-EXE-Datei.

Das Download-Archiv enthält neben dem eigentlichen EUKLID DynaGeo-Programm noch eine Serie von Beispielzeichnungen sowie die Viewer "DynaGeoX" und "DynaGeoJ". Was davon Sie installieren wollen, können Sie zunächst in einem Auswahl-Dialog festlegen. Wenn Sie Wert darauf legen, dass das Installationsprogramm möglichst wenig Einträge in die WINDOWS-Registry macht, sollten Sie weder den DynaGeoX-Viewer installieren noch die Datei-Typen ".GEO" (bzw. das veraltete ".GEOX") für DynaGeo.exe registrieren lassen. Dann werden nur ein paar Registry-Einträge gemacht, die WINDOWS braucht, um die eventuelle spätere De-Installation des Programms zu ermöglichen.

Bei der lokalen Installation auf einem Einzelarbeitsplatz wird man jedoch in der Regel eine Voll-Installation machen: lassen Sie beide Viewer installieren und die DynaGeo-Datei-Typen in WINDOWS registrieren, dann können Sie EUKLID DynaGeo mit all seinen Möglichkeiten komfortabel nutzen!
