Menü Datei

Das Datei-Menü ermöglicht das Abspeichern, Öffnen und Ausdrucken von EUKLID DynaGeo - Zeichnungen. Im einzelnen enthält das Menü die folgenden Einträge:


    Neu

    Öffnen...
    Speichern
    Speichern unter...
    Als DynaGeoX-HTML-Seite speichern...
    Als DynaGeoJ-HTML-Seite speichern...

    Drucker-Optionen
    Druckbild-Vorschau...
    Drucken

    Ende

Normalerweise enthält dieses Menü oberhalb des Menüpunktes Ende außerdem eine Liste der letzten bearbeiteten GEO-Dateien. Ob eine solche Liste angezeigt wird und wie lang sie maximal sein kann, können Sie unter Verschiedenes | Einstellungen auf der Registerkarte Menükonfiguration einstellen.
