Letzte Löschung widerrufen    

Mit diesem Befehl können Sie die letzte Löschung widerrufen und so das zuletzt gelöschte Objekt wieder herstellen. Dabei ist es egal, ob das Objekt durch Einen Konstruktionsschritt zurück oder Objekt löschen gelöscht wurde. Der Widerrufen-Befehl wirkt so, als hätten Sie das gelöschte Objekt neu erzeugt.


Achtung !

    Auch wenn bei Ihrer letzten Löschaktion mit Objekt löschen eine ganze Reihe von Objekten auf einen Schlag gelöscht wurden, bezieht sich der Widerrufen-Befehl stets nur auf ein einziges Objekt, nämlich das zuletzt gelöschte. Um mehrere Objekte wieder herzustellen, müssen Sie den Befehl entsprechend oft anwenden.
