Bindung einer Textbox lösen

Klicken Sie mit der linken Maustaste auf die Textbox, deren Bindung an ein Referenz-Objekt Sie lösen wollen. Zum Zeichen, dass die Bindung erfolgreich gelöst wurde, rückt die Textbox etwas von diesem Objekt weg.

Danach wird der Text nicht mehr der Bewegung dieses Objektes folgen, sondern an seinem Ort in der Zeichnung verbleiben. Nach wie vor können Sie die Textbox aber mit der Maus an eine von Ihnen gewünschte Stelle verschieben.
