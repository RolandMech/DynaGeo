Speichern unter...

Es wird ein Dialogfenster geöffnet, in dem Sie den (neuen) Namen der Datei eingeben können. Zu Ihrer besseren Orientierung wird auch hier eine Liste der schon vorhandenen GEO-Dateien angezeigt, aus der Sie auch einen Namen auswählen können. Stattdessen können Sie jedoch auch einen neuen Namen direkt ins Editierfeld Dateiname eingeben. Nach Ihrer Bestätigung (Okay) wird die aktuelle Zeichnung in diese Datei geschrieben. Existiert schon eine Datei mit diesem Namen, werden Sie gefragt, ob sie überschrieben werden soll.

Nach dem Abspeichern erscheint der neue Dateiname nun als der aktuelle Name Ihrer Zeichnung in der Kopfzeile.

Sie können in diesem Dialogfenster eine EUKLID DynaGeo - Zeichnung auch als Grafik-Datei abspeichern. Verfügbar sind dafür die Pixelformate BMP-, JPG- und PNG-, sowie die Vektorformate WMF- und EMF. Wählen Sie dazu in der Liste Dateityp (unten) die passende Kennung des Formats. Dann wird Ihnen automatisch eine Liste der schon vorhandenen Grafik-Dateien dieses Typs angezeigt. Nach Eingabe bzw. Auswahl eines Dateinamens und einem Klick auf Okay wird die Grafik-Datei geschrieben. Falls Sie vor dem Aufruf des Dialogfensters einen Auswahlrahmen aufgezogen haben, wird nur der entsprechende Teil der Zeichnung in die Grafik-Datei geschrieben.


Bemerkungen:

  • Wenn Sie die exportierte Grafik übers Netz versenden oder in einer Web-Seite verwenden wollen, ist eine möglichst kompakte Pixelgrafik vorteilhaft. In dieser Beziehung ist das PNG-Format zu empfehlen, das die derzeit wohl komprimierteste aller verfügbaren verlustlosen Kodierungen aufweist und entsprechend zu den kleinsten Dateien führt.

  • Wenn Sie beim Export einer Zeichnung als Grafik im Hinblick auf die spätere Verwendung möglichst keine Qualitätsverluste haben wollen, dann können Sie Ihre Konsturktion in einem Vektorformat (vorzugsweise im EMF-Format) abspeichern. Wenn Sie z.B. die Grafik in WORD einbinden und das Dokument später ausdrucken, dann wird eine Pixelgrafik im Ausdruck immer nur diejenige Auflösung zeigen, die sie zum Zeitpunkt des Abspeicherns auf dem Bildschirm hatte; eine Vektorgrafik hingegen kann mit der vollen Auflösung des Druckers aufs Papier gebracht werden.

  • Wenn eine exportierte EMF-Grafik im aufnehmenden Programm jedoch nicht korrekt dargestellt wird, weil dieses Programm die Vektorgrafik teilweise nicht richtig "versteht", dann können Sie die Konstruktion doch in einem Pixelformat exportieren, aber die Auflösung der Grafikdatei höhersetzen: dazu wählen Sie unter "Einstellungen | Export" die gewünschte Auflösung für den Export von Pixel-Bildern aus der dort angebotenen Liste aus. Damit steht dann später (z.B. beim Ausdrucken) eine höher aufgelöste Grafik zur Verfügung, was gar zu schlimme Qualitätsverluste vermeiden hilft.

  • Das früher noch verfügbare GEOX-Format ist inzwischen veraltet und wird nicht mehr zum Abspeichern von Zeichnungen verwendet. EUKLID DynaGeo kann allerdings nach wie vor alte Konstruktionen aus GEOX-Dateien einlesen und weiterbearbeiten. Beim Abspeichern der geänderten Zeichnung müssen Sie dann allerdings das GEO-Format benutzen. Falls die Zeichnung von einer schon bestehenden Web-Seite benutzt werden soll, muss in dieser der Dateiname im "<object>"-Tag entsprechend angepasst werden.
